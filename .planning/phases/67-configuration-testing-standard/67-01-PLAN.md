
   ```python
   @pytest.fixture
   def sample_dataframe_factory():
       def _create(n_rows=100, columns=None, seed=42):
           # Factory function
       return _create
   ```
3. Fixture scope guidelines (session, module, function)
4. fixtures/ directory for test data files
5. factories/ directory for test data generators
