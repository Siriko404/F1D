  <action>
    Replace the sequential year loop with ProcessPoolExecutor:

    Add import at top:
    ```python

    ```

    Replace main() year loop with:
    ```python
    def main():
        # ... existing setup code ...

        years = range(2002, 2019)
        thread_count = config["determinism"]["thread_count"]

        print(f"\nProcessing {len(years)} years with {thread_count} worker(s)...")

        results = {}
        with ProcessPoolExecutor(max_workers=thread_count) as executor:
            # Submit all years
            futures = {
                executor.submit(
                    process_year_worker,
                    year,
                    root,
                    config,
                    valid_files,
                    vocab_list,
                    cat_sets,
                    out_dir
                ): year
                for year in years
            }

            # Collect results as they complete
            for future in as_completed(futures):
                year = futures[future]
                try:
                    _, year_stats = future.result()
                    results[year] = year_stats
                except Exception as e:
                    print(f"  ERROR: Year {year} failed: {e}")
                    raise

        # Combine stats in year order for determinism
        all_stats = {
            "years": [results[y] for y in sorted(results.keys())],
            "total_input_rows": sum(s.get("input_rows", 0) for s in results.values()),
            "total_output_rows": sum(s.get("output_rows", 0) for s in results.values()),
            "total_filtered_rows": sum(s.get("filtered_rows", 0) for s in results.values()),
            "total_tokens": sum(s.get("total_tokens", 0) for s in results.values()),
            # ... other aggregate stats ...
        }
    ```

    Key requirements:
    - Use as_completed for non-blocking collection
    - Track year for each future (dictionary mapping)
    - Sort results by year before aggregating (determinism)
    - Handle exceptions and propagate errors
