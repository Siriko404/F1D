  <action>
Move 2_Scripts/ARCHIVE_OLD/ to .___archive/legacy/ARCHIVE_OLD/ using git mv for tracked files:

1. Check git tracking status first and move file-by-file with conditional logic:
```bash
# Create destination directory
mkdir -p .___archive/legacy/

# Track counts for verification
tracked_count=0
untracked_count=0

cd 2_Scripts/ARCHIVE_OLD
for file in *; do
    if [ -e "$file" ]; then
        # Check if file is tracked by git
        if git ls-files --error-unmatch "$file" >/dev/null 2>&1; then
            echo "TRACKED: $file - using git mv"
            git mv "$file" "../../.___archive/legacy/ARCHIVE_OLD/"
            tracked_count=$((tracked_count + 1))
        else
            echo "UNTRACKED: $file - using regular mv"
            mv "$file" "../../.___archive/legacy/ARCHIVE_OLD/"
            untracked_count=$((untracked_count + 1))
        fi
    fi
done
cd ../..

echo "Move summary: $tracked_count tracked files (git mv), $untracked_count untracked files (mv)"
```
