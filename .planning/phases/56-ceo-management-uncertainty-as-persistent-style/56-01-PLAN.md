Create new script 2_Scripts/4_Econometric_V2/4.9_CEOFixedEffects.py implementing Dzieliński et al. (2020) CEO fixed effects extraction:

**Equation 4 Specification:**
```
UncAns_i,t = alpha + gamma_i*CEO_i + Sum_s(beta_s*Speech_s) + Sum_k(beta_k*FirmChars_k) + Year_t + epsilon_i,t
```

**Variable Mapping (V2 to Paper):**
- Dependent: CEO_QA_Uncertainty_pct (UncAns - CEO Q&A uncertainty only, LM dictionary)
- Speech controls: CEO_Pres_Uncertainty_pct (UncPreCEO), Analyst_QA_Uncertainty_pct (UncQue), CEO_All_Negative_pct (NegCall)
- Firm controls: SurpDec, EPS_Growth, StockRet, MarketRet
- Fixed effects: C(ceo_id) for CEO FE, C(year) for Year FE
- Standard errors: cov_type='cluster', cov_kwds={'groups': ceo_id} (CEO-clustered per paper)

**Key Implementation Requirements:**
1. Follow V1 pattern from 4.1_EstimateCeoClarity.py lines 338-452 for FE extraction
2. Use statsmodels Formula API with C(ceo_id) for CEO dummies (LSDV method)
3. Filter to CEOs with >= 5 calls (paper requirement, page 45)
4. Run BOTH sample periods in parallel: paper (2003-2015) and extended (2002-2018)
5. ClarityCEO_i = -gamma_i (negate FIRST), THEN standardize to mean=0, SD=1
6. Handle reference CEO (omitted from params) with gamma=0
7. DO NOT include firm fixed effects (only in turnover analysis, out of scope)

**Data Loading:**
- Use get_latest_output_dir() for timestamp-based directory resolution
- Load manifest: 4_Outputs/1.4_AssembleManifest/latest/master_sample_manifest.parquet (columns: file_name, gvkey, ceo_id, ceo_name)
- Load linguistic: 4_Outputs/2_Textual_Analysis/2.2_Variables/latest/linguistic_variables_{year}.parquet
- Load controls: 4_Outputs/3_Financial_Features/latest/firm_controls_{year}.parquet and market_variables_{year}.parquet

**Table 3 Specifications (Primary):**
- Column (1) Baseline: UncAns ~ CEO FE + UncPreCEO + UncQue + Year FE
- Column (2) Full Equation 4: UncAns ~ CEO FE + UncPreCEO + UncQue + NegCall + SurpDec + EPS_Growth + StockRet + MarketRet + Year FE
- Report R-squared, adjusted R-squared, F-statistics, coefficients, t-statistics
- Benchmark R2 = 0.31 (CEO FE only), report delta R2 vs benchmark

**Table IA.1 Robustness Specifications (5 specs only - V2 data limitation):**
- Spec (0): CEO FE only
- Spec (1): + UncPreCEO
- Spec (2): + UncPreCEO + Firm chars
- Spec (3): Baseline (Eq. 4 = Table 3 Col 1) - reference
- Spec (7): + delta_UncPreCEO (change in CEO presentation uncertainty)
- SKIP specs (4), (5), (6) - CFO, EPR, AnDispPre not available in V2 (verified in 56-RESEARCH.md Pitfall 3: "V2 text analysis doesn't tag CFO speech separately")
- Output: 5x5 correlation matrix of CEO FE across specs (specs 0,1,2,3,7 only)

**delta_UncPreCEO Computation (spec 7):**
- Formula: delta_UncPreCEO_i,t = UncPreCEO_i,t - UncPreCEO_i,t-1
- Implementation: Group by ceo_id, apply lag(1) to CEO_Pres_Uncertainty_pct, compute difference
- Creates: CEO-quarter level change variable capturing within-CEO evolution of presentation uncertainty

**Output Files:**
- ceo_clarity_scores.parquet: ceo_id, ceo_name, gamma_i, ClarityCEO_raw, ClarityCEO, n_calls, sample_period, specification
- table3_replication.csv: Coefficients, t-stats, R2, F-stats for both periods
- table_ia1_correlation_matrix.csv: 5x5 Correlation of CEO FE across specs (0,1,2,3,7), baseline (spec 3) = 1.00
- clarity_distribution_histogram.png: Figure 3 replication with mean/SD annotations
- regression_results_{period}_{spec}.txt: Full model output
- report_step56.md: Summary report

**Script Structure:**
- Imports: sys, os, pathlib, pandas, numpy, statsmodels.formula.api, matplotlib
- CLI: --dry-run for validation
- load_data(): Load and merge manifest, linguistic, firm/market controls per year
- prepare_regression_data(): Filter to complete cases, apply >=5 calls filter
- run_ceo_fe_regression(): Run OLS with C(ceo_id) + C(year), extract gamma_i
- extract_ceo_fixed_effects(): Parse CEO dummy coefficients, compute ClarityCEO
- run_robustness_specs(): Run 5 Table IA.1 specifications, compute correlations
- compute_delta_uncpreceo(): Lagged difference: UncPreCEO_i,t - UncPreCEO_i,t-1 grouped by ceo_id
- create_figure3_histogram(): Plot ClarityCEO_raw distribution
- create_table3_replication(): Output regression coefficients and diagnostics
- generate_report(): Summary with sample statistics, distribution, F-stats
- main(): Orchestrate execution for both periods

**Anti-Patterns to Avoid:**
- DO NOT use linearmodels PanelOLS (harder to extract individual gamma_i)
- DO NOT cluster by firm (paper clusters by CEO)
- DO NOT use HC1 SE (paper requires CEO clustering)
- DO NOT forget reference CEO with gamma=0
- DO NOT use CEO_All_Uncertainty_pct or Manager_QA_Uncertainty_pct (must be CEO_QA_Uncertainty_pct only)
- DO NOT attempt specs 4-6 (CFO, EPR, AnDispPre) - these variables do not exist in V2 data
