## Design Context (from 42-CONTEXT.md - LOCKED)

### Model Specification (Primary)
```
Uncertainty_{i,t} = beta0 + beta1*CCCL_{i,t-1} + gamma*Controls_{i,t}
                   + Firm_FE + Year_FE + epsilon_{i,t}
```

Where:
- Uncertainty: One of 6 measures (Manager/CEO × QA/Pres × Uncertainty/Weak_Modal)
- CCCL_{t-1}: Lagged shift-share instrument (primary: shift_intensity_mkvalt_ff48)
- Controls: Size, Leverage, ROA (time-varying)
- FE: Firm + Year (NO Industry × Year - absorbs treatment variation)
- SE: Clustered at firm-year level (4 quarters share CCCL exposure)

### Hypotheses
- **H6-A:** beta1 < 0 (CCCL reduces uncertainty - primary)
- **H6-B:** |beta1_QA| > |beta1_Pres| (stronger in spontaneous Q&A - mechanism)
- **H6-C:** CCCL reduces Uncertainty_Gap (consistency test)

### Identification Requirements
1. **FDR Correction:** Benjamini-Hochberg across 6 uncertainty measures
2. **Pre-trends:** Test CCCL_{t-2}, CCCL_{t-1} leads (should be insignificant)
3. **Robustness:**
   - Firm-specific linear trends (firms with ≥10 years)
   - Alternative FE specifications
   - Lagged DV specification (Nickell bias check)
   - Alternative CCCL variants (6 instruments)

### Literature Controls for Q&A Specifications
When testing Q&A uncertainty, control for:
- Pres_Uncertainty (scripted component)
- Analyst_Uncertainty (conversation partner effect)

### Established Patterns (from 4.5_H5DispersionRegression.py)
- Use `run_panel_ols()` from shared.panel_ols for all regressions
- Use `get_latest_output_dir()` to resolve H6 variables input
- Specification dictionary for FE variants
- One-tailed p-values for directional hypotheses
- VIF check before regression (multicollinearity diagnostics)
- Cluster SE at appropriate level (firm-year for CCCL)
- Results stored as list of dictionaries, converted to DataFrame
- Hierarchical output: parquet + markdown + stats.json
</context>
