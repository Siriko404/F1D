## Design Context (from 42-CONTEXT.md - LOCKED)

### Shift-Share Construction
- **Primary Instrument:** `shift_intensity_mkvalt_ff48` (FF48 industry × market value, normalized 0-1)
- **Robustness variants:** FF48/Sales, FF12/Market Value, FF12/Sales, SIC2/Market Value, SIC2/Sales
- **Treatment timing:** Lagged (t-1 predicts t) to avoid reverse causality
- **FE Strategy:** Firm FE + Year FE (NO Industry × Year FE - would absorb treatment variation)

### Speech Uncertainty DVs (6-measure suite)
1. Manager_QA_Uncertainty_pct (Primary)
2. Manager_QA_Weak_Modal_pct (Secondary)
3. CEO_QA_Uncertainty_pct (Robustness)
4. CEO_QA_Weak_Modal_pct (Robustness)
5. Manager_Pres_Uncertainty_pct (Robustness)
6. CEO_Pres_Uncertainty_pct (Robustness)
- Additional: Uncertainty_Gap = Manager_QA - Manager_Pres

### Data Sources
- Treatment: `1_Inputs/CCCL instrument/instrument_shift_intensity_2005_2022.parquet` (145K firm-years, 6 variants)
- Outcome: `4_Outputs/2_Textual_Analysis/2.2_Variables/latest/linguistic_variables_*.parquet` (2002-2018)
- Merge key: gvkey + fiscal_year + fiscal_quarter
- Sample period: 2005-2018 overlap
- Expected: ~200-400K firm-quarter observations

### Established Patterns (from 3.5_H5Variables.py)
- Use `get_latest_output_dir()` for timestamp-based path resolution
- Use `DualWriter` for mirrored stdout/log output
- Use `validate_input_file()` for input validation
- Read CCCL instrument via pandas (not PyArrow - smaller file)
- Read linguistic variables via chunked concatenation across years
- Aggregate multiple calls per quarter using mean
- Create lagged treatment via groupby-shift pattern
- Standardize gvkey to 6-digit zero-padded string
