<task type="auto">
  <name>Task 1: Create configuration loader with caching</name>
  <files>src/f1d/shared/config/loader.py</files>
  <action>
Create configuration loader module with caching and error handling:

1. **ConfigError** Exception:
   - Custom exception for configuration-related errors
   - Includes path to config file and validation message

2. **ConfigCache** class (internal):
   - _config: Optional[ProjectConfig] = None
   - _config_path: Optional[Path] = None
   - _loaded_at: Optional[float] = None

3. **get_config(path: Path | None = None, reload: bool = False) -> ProjectConfig**:
   - Returns cached config if available and path matches
   - Loads from path if not cached or reload=True
   - Raises ConfigError with clear message if validation fails
   - Handles FileNotFoundError with helpful message

4. **reload_config() -> ProjectConfig**:
   - Forces reload of configuration
   - Clears cache and reloads from same path

5. **clear_config_cache() -> None**:
   - Clears the cached configuration (useful for testing)

Example usage:
```python
from f1d.shared.config import get_config

config = get_config()  # Loads from config/project.yaml
print(config.data.year_start)

# Environment override: F1D_DATA__YEAR_START=2005
# config.data.year_start == 2005
```
