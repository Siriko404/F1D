<objective>
**Objective:** Merge PRisk (political risk) and Managerial Uncertainty measures with InvestmentResidual, standardize both components, and create the interaction term PRisk_x_Uncertainty as the independent variable.

**Purpose:** Construct the complete regression dataset for testing H2: whether compound uncertainty (PRisk x Uncertainty) predicts decreased investment efficiency. This plan merges three data sources (PRisk, Uncertainty, InvestmentResidual), standardizes the interaction components, and creates all required variables for regression.

**Output:**
- `4_Outputs/3_Financial_V3/4.2_H2_PRiskUncertaintyMerge/{timestamp}/H2_PRiskUncertainty_Analysis.parquet`: Complete regression dataset with IV (PRisk_x_Uncertainty), DV (InvestmentResidual), main effects (PRisk_std, Uncertainty_std), and all Biddle (2009) controls
- `4_Outputs/3_Financial_V3/4.2_H2_PRiskUncertaintyMerge/{timestamp}/stats.json`: Merge statistics, standardization validation, correlation matrix
- `3_Logs/3_Financial_V3/4.2_H2_PRiskUncertaintyMerge/{timestamp}_Merge.log`: Execution log

**Key Specification:**
- Standardize BEFORE creating interaction: PRisk_std = (PRisk - mean)/sd, Uncertainty_std = (Uncertainty - mean)/sd
- Interaction: PRisk_x_Uncertainty = PRisk_std * Uncertainty_std
- Rationale: Standardization yields interpretable coefficients (one SD change) and reduces multicollinearity concerns
</objective>

**create_interaction_term():**
```
PRisk_x_Uncertainty = PRisk_std * Uncertainty_std
```
- Multiply standardized components (NOT raw variables)
- Do NOT include main effects in interaction calculation (standardization handles this)

**validate_interaction():**
- Report correlation matrix: PRisk_std, Uncertainty_std, PRisk_x_Uncertainty
- Check for extreme multicollinearity (VIF > 10 is concerning)
- Document that standardization (not mean-centering) is used per hypothesis specification
- Rationale: Standardization yields interpretable coefficients (1 SD change) and reduces multicollinearity

**Note on mean-centering:**
Research document confirms mean-centering does NOT reduce multicollinearity for interaction terms (it's an expected mathematical property). Standardization is used for interpretability, not multicollinearity reduction.
