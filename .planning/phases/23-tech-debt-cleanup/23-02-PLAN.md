## Utility Functions

The following utility functions are available across shared modules:

### Logging and Output
**Module:** `shared.observability_utils`

- **`DualWriter`** - Class for dual-write logging (stdout + log file)
- **`compute_file_checksum(filepath, algorithm="sha256")`** - Compute file checksum (SHA-256 by default)
- **`print_stat(label, before=None, after=None, value=None, indent=2)`** - Print formatted statistics
- **`print_stats_summary(stats)`** - Print formatted summary table
- **`save_stats(stats, out_dir)`** - Save statistics dictionary to JSON

### Data Analysis
**Module:** `shared.observability_utils`

- **`analyze_missing_values(df)`** - Analyze missing values per column (returns dict with count/percent)
- **`get_process_memory_mb()`** - Get current process memory usage (rss_mb, vms_mb, percent)
- **`calculate_throughput(rows_processed, duration_seconds)`** - Calculate rows/second throughput
- **`detect_anomalies_zscore(df, columns, threshold=3.0)`** - Detect outliers using z-score method
- **`detect_anomalies_iqr(df, columns, multiplier=3.0)`** - Detect outliers using IQR method

### Symlink and Link Management
**Module:** `shared.symlink_utils`

- **`update_latest_link(target_dir, link_path, verbose=True)`** - Update 'latest' symlink/junction
  - Unix: Uses symbolic links
  - Windows: Tries symlink, falls back to junction, then copy
  - Handles permissions gracefully with detailed error messages
- **`create_junction(target, link_path)`** - Create Windows junction (directory link)
- **`is_junction(path)`** - Check if path is a Windows junction

### Usage Example

```python
from shared.observability_utils import (
    DualWriter,
    compute_file_checksum,
    print_stat,
    analyze_missing_values,
)
from shared.symlink_utils import update_latest_link

# Set up dual-write logging
dw = DualWriter("path/to/log.txt")
dw.write("Processing started\n")

# Compute checksum for reproducibility
checksum = compute_file_checksum(output_path)

# Print statistics with formatting
print_stat("Rows processed", value=1000)

# Analyze missing data
missing = analyze_missing_values(df)

# Update latest symlink
update_latest_link(output_dir, latest_dir)
```

### Module Reference

- **`shared.observability_utils`** - Core statistics and logging utilities
- **`shared.symlink_utils`** - Cross-platform symlink/junction management
- **`shared.path_utils`** - Path validation and directory creation
- **`shared.data_validation`** - Input data validation layers
```
