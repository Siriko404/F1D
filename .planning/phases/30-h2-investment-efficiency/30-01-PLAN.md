Create the H2 variable construction script following the established 3.1_H1Variables.py pattern. The script must:

**Header and Imports:**

**Configuration and Paths:**

- Output to `4_Outputs/3_Financial_V2/{timestamp}/H2_InvestmentEfficiency.parquet`
- Log to `3_Logs/3_Financial_V2/{timestamp}_H2.log`

**Data Loading Functions:**
- `load_manifest()` - same as H1
- `load_compustat_h2()` - PyArrow schema inspection, load columns: gvkey, datadate, fyearq, sic, capxy, dpq, saleq, saley, atq, iby, cheq, ceqq, cshoq, prccq, dlttq, dlcq, oancfy
- `load_ibes()` - Load tr_ibes.parquet with columns: cusip, fpedats, fiscalp, numest, meanest, stdev
- `build_ff_mappings()` - Parse Siccodes48.zip and Siccodes12.zip using parse_ff_industries()

**Industry Classification:**
- `assign_ff_industries(df, ff48_map, ff12_map)` - Add ff48 and ff12 columns, unmapped SIC -> 48 (Other)

**Over/Underinvestment Classification (H2-01, H2-02):**
- `compute_capex_dp(df)` - Capex/Depreciation with dp > 0 guard: `df["dpq"].where(df["dpq"] > 0)`
- `compute_sales_growth(df)` - (sale_t - sale_{t-1}) / |sale_{t-1}| with groupby("gvkey").shift(1)
- `compute_industry_year_median(df, value_col)` - FF48 groupby, fallback to FF12 if cell < 5 firms
- `compute_overinvest_dummy(df)` - 1 if capex_dp > 1.5 AND sales_growth < ind_year_median
- `compute_underinvest_dummy(df)` - 1 if capex_dp < 0.75 AND tobins_q > 1.5
- Enforce mutual exclusivity: if both flags set, set both to 0 with warning

**Efficiency Score (H2-03):**
- `compute_efficiency_score(df)` - Rolling 5-year window (t-4 to t), min 3 years required
- Formula: 1 - (# inefficient years / # years in window)
- inefficient = overinvest_dummy | underinvest_dummy

**Biddle ROA Residual (H2-04):**
- `compute_delta_roa(df)` - ROA(t+2) - ROA(t) using groupby("gvkey")["roa"].shift(-2)
- `compute_roa_residuals(df)` - For each FF48-year with >= 20 obs, run OLS:
  - Y: delta_roa_t2
  - X: capex_at, tobins_q, cash_holdings, leverage (add constant)
  - Extract residual = actual - predicted
  - Fall back to FF12 if FF48 cell < 20 obs
  - Return DataFrame with index, roa_residual, predicted_delta_roa

**IBES Analyst Dispersion (H2-05):**
- `link_ibes_to_compustat(ibes_df, ccm_file)` - Match on CUSIP8 (first 8 chars)
- `compute_analyst_dispersion(ibes_linked)` - STDEV / |MEANEST|
  - Filter: numest >= 2, |meanest| >= 0.01
  - Aggregate to firm-year level (median dispersion)

**Control Variables (H2-05, H2-06):**
- `compute_tobins_q(df)` - (AT + ME - CEQ) / AT (reuse from H1)
- `compute_cf_volatility(df)` - 5-year rolling StdDev of OANCF/AT (similar to H1's ocf_volatility)
- `compute_industry_capex_intensity(df)` - Mean(capex_at) by FF48-year
- `compute_firm_size(df)` - ln(AT) (reuse from H1)
- `compute_roa(df)` - IB/AT (reuse from H1)
- `compute_fcf(df)` - (OANCF - CAPX) / AT
- `compute_earnings_volatility(df)` - 5-year rolling StdDev of ROA (proxy for earnings vol)

**Merge and Output:**
- Build base from unique gvkey-fiscal_year in Compustat
- Merge all computed variables
- Merge with manifest to filter to sample
- Apply winsorization at 1%/99% for continuous variables
- Compute variable statistics for stats.json
- Sort by gvkey, fiscal_year for determinism
- Write H2_InvestmentEfficiency.parquet and stats.json

**Main Function:**
- Parse arguments (--config, --output-dir, --dry-run)
- Check prerequisites
- Setup DualWriter logging
- Execute full pipeline with timing and memory tracking
- Print summary with observation counts

**Key Implementation Notes:**
- Use PyArrow schema inspection BEFORE reading full Compustat (avoid OOM)
- gvkey normalization: str, zfill(6)
- Handle edge cases: DP=0, near-zero MEANEST, thin industry cells
- Log counts at each step for debugging
- Follow exact variable naming from README.md
