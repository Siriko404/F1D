2. **`run_panel_ols()` function** (main entry point):
   ```python
   def run_panel_ols(
       df: pd.DataFrame,
       dependent: str,
       exog: List[str],
       entity_col: str = 'gvkey',
       time_col: str = 'year',
       industry_col: str = 'ff48_code',
       entity_effects: bool = True,
       time_effects: bool = True,
       industry_effects: bool = False,  # Default False - firm FE subsumes industry
       cov_type: str = 'clustered',     # 'clustered', 'kernel', 'robust'
       cluster_cols: Optional[List[str]] = None,  # For double-clustering
       kernel: str = 'bartlett',        # For HAC adjustment (ECON-06)
       min_industry_firms: int = 5,     # Auto-collapse thin cells to FF12
       check_collinearity: bool = True,
       vif_threshold: float = 5.0
   ) -> Dict[str, Any]:
   ```

3. **Fixed effects handling** (ECON-01):
   - Set MultiIndex: `df.set_index([entity_col, time_col])`
   - Use `PanelOLS(entity_effects=True, time_effects=True)`
   - For industry FE: use `other_effects=df[industry_col]` (warn about FE collinearity trap)
   - Check for thin industry-year cells: if any cell has < `min_industry_firms` firms, fall back from FF48 to FF12 and log warning

4. **Clustered SE** (ECON-03):
   - Single clustering: `model.fit(cov_type='clustered', cluster_entity=True, debiased=True)`
   - Double clustering: Pass cluster array to `clusters` parameter
   - HAC: `model.fit(cov_type='kernel', kernel='bartlett', bandwidth=None)` (auto bandwidth)

5. **Collinearity detection**:
   - Set `drop_absorbed=False, check_rank=True` on PanelOLS
   - If `check_rank` fails (perfect collinearity), raise `CollinearityError` with informative message
   - Call `compute_vif()` from diagnostics.py post-fit; if any VIF > threshold, log bold warning

6. **Return dictionary** with:
   - `model`: fitted model object
   - `coefficients`: DataFrame with beta, SE, t-stat, p-value per variable
   - `summary`: dict with R2, adj_R2, N, F-stat, fixed_effects_used
   - `diagnostics`: dict with VIF values, condition_number, residual_std
   - `warnings`: list of any warnings triggered

7. **Console output**: Print aligned summary table with significance stars (*, **, *** for 10%, 5%, 1%)


   ```python
   def compute_vif(
       df: pd.DataFrame,
       columns: List[str],
       add_constant: bool = True
   ) -> pd.DataFrame:
   ```
   - Use `statsmodels.stats.outliers_influence.variance_inflation_factor()`
   - Only compute VIF on continuous regressors (exclude dummies, FE columns)
   - Return DataFrame with columns: variable, VIF, threshold_exceeded (bool)
   - Handle edge cases: drop rows with any NaN before computing

3. **`check_multicollinearity()` function**:
   ```python
   def check_multicollinearity(
       df: pd.DataFrame,
       columns: List[str],
       vif_threshold: float = 5.0,
       condition_threshold: float = 30.0,
       fail_on_violation: bool = False
   ) -> Dict[str, Any]:
   ```
   - Compute VIF for all columns
   - Compute condition number of design matrix
   - Return dict with:
     - `vif_results`: DataFrame from compute_vif()
     - `condition_number`: float
     - `vif_violations`: list of columns with VIF > threshold
     - `condition_violation`: bool (condition_number > threshold)
     - `pass`: bool (no violations)
   - If `fail_on_violation=True` and any violation, raise `MulticollinearityError`

4. **`format_vif_table()` function**:
   ```python
   def format_vif_table(vif_df: pd.DataFrame, threshold: float = 5.0) -> str:
   ```
   - Return formatted string table for console output
   - Highlight violations in bold (use ANSI codes or asterisks for Windows compatibility)
   - Show "VIF 4.2 -- acceptable, below 5 threshold" format
