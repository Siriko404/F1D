**Clean up existing symlink/junction/copy directories:**

Run commands to identify and remove all `latest/` directories or symlinks in 4_Outputs/:

```bash
# Find all latest directories/symlinks
find 4_Outputs -name "latest" -type d -o -name "latest" -type l 2>/dev/null

# Remove them (on Windows, may need different approach)
# Python script for cross-platform removal:
python -c "
import shutil
from pathlib import Path

outputs = Path('4_Outputs')
for p in outputs.rglob('latest'):
    if p.is_symlink() or p.is_junction() if hasattr(p, 'is_junction') else False:
        p.unlink()
        print(f'Removed symlink: {p}')
    elif p.is_dir():
        shutil.rmtree(p)
        print(f'Removed directory: {p}')
"
```
