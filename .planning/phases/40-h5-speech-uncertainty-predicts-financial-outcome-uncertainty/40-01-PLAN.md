Create `3.5_H5Variables.py` following the pattern of `3.2_H2Variables.py` but with H5-specific logic:

**Script Header:**
```
==============================================================================
STEP 3.5: H5 Analyst Dispersion Variables
==============================================================================
ID: 3.5_H5Variables
Description: Constructs H5 dependent variable (analyst forecast dispersion at t+1),
             control variables, and uncertainty gap measure for testing whether
             hedging language predicts analyst disagreement.
```

**Key Implementation Details:**

1. **Analyst Dispersion (Refined):**
   - Load IBES `tr_ibes.parquet` with columns: CUSIP, FPEDATS, FISCALP, NUMEST, MEANEST, STDEV, ACTUAL
   - Filter: `NUMEST >= 3` (standard literature threshold)
   - Filter: `|MEANEST| >= 0.05` (prevents ratio explosion)
   - Compute: `dispersion = STDEV / |MEANEST|`
   - Winsorize at 1%/99%
   - Extract `fiscal_quarter` from FPEDATS for quarterly matching

2. **Forward-Looking Timing (Critical):**
   - Match speech at quarter t to dispersion at quarter t+1
   - Use `groupby('gvkey').shift(-1)` on dispersion to create `dispersion_lead`
   - This is the DV: `dispersion_lead` (next quarter's dispersion)

3. **Prior Dispersion (Lagged DV):**
   - Create `prior_dispersion` as current quarter dispersion (before shift)
   - This controls for persistence (~0.6-0.8 autocorrelation)

4. **Earnings Surprise:**
   - Compute: `earnings_surprise = |ACTUAL - MEANEST| / |MEANEST|`
   - Use same |MEANEST| >= 0.05 filter
   - This is determined BEFORE speech (confounding control)

5. **Loss Dummy:**
   - From Compustat: `loss_dummy = 1 if NI < 0`
   - Merge via gvkey-fiscal_quarter

6. **Uncertainty Gap (H5-B):**
   - Load speech measures from Step 2 linguistic_variables
   - Compute: `uncertainty_gap = Manager_QA_Uncertainty_pct - Manager_Pres_Uncertainty_pct`
   - Gap > 0 means more uncertain in spontaneous speech

7. **CCM CUSIP-GVKEY Linking:**
   - Load CCM file (1_Inputs/ccm/ccm.parquet or similar)
   - Match IBES CUSIP8 to Compustat GVKEY
   - If CCM not available, use direct CUSIP8 matching from Compustat header

8. **Control Variables (merge from existing):**
   - firm_size (from H1/H2 outputs or recompute)
   - leverage (from H1 outputs)
   - tobins_q (from H2 outputs)
   - earnings_volatility (from H2 outputs)
   - analyst_coverage (NUMEST - log transform)
