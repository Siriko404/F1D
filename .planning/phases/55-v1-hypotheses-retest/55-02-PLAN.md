  <files>.planning/phases/55-v1-hypotheses-retest/55-METHODOLOGY.md</files>
  <action>
1. Create section "## Hypothesis 1: Managerial Speech Uncertainty -> Stock Illiquidity" in 55-METHODOLOGY.md

2. Specify hypothesis:
   - H7a: Higher speech uncertainty predicts HIGHER stock illiquidity (worse liquidity)
   - Direction: beta_Uncertainty > 0 (uncertainty increases illiquidity measure)

3. Specify dependent variables:
   - **Primary:** Amihud (2002) illiquidity ratio
     * Formula: ILLIQ_{i,t} = (1/D_{i,t}) * sum(|r_{i,t,d}| / VOLD_{i,t,d})
     * Where: r = daily return, VOLD = daily dollar volume, D = trading days
     * Calculation at firm-year level
     * Transform: log(1 + ILLIQ) or use raw scaled by 1e6

   - **Robustness 1:** Roll (1984) implicit spread
     * Based on price autocovariance
     * SPRD = 2 * sqrt(-cov(r_t, r_{t-1}))

   - **Robustness 2:** Bid-ask spread (if available in data)
     * Effective spread from CRSP

4. Specify independent variable:
   - Reuse V2 uncertainty measures (per CONTEXT decision):
     * Manager_QA_Uncertainty_pct
     * CEO_QA_Uncertainty_pct
     * Manager_Pres_Uncertainty_pct
     * CEO_Pres_Uncertainty_pct
   - Aggregation: Annual average at firm-year level

5. Specify control variables (from literature):
   - Size: log(Market Cap) or log(Assets)
   - Leverage: (DLTT + DLC) / AT
   - ROA: IB / AT
   - MTB: Market-to-book ratio
   - Volatility: Stock return volatility (annual std dev)
   - Stock Returns: Cumulative returns
   - Current Ratio: ACT / LCT

6. Specify primary regression equation:
   ```
   Illiquidity_{i,t+1} = beta0 + beta1*Uncertainty_{i,t}
                        + gamma*Controls_{i,t}
                        + alpha_i + delta_t + epsilon_{i,t}
   ```
   Where:
   - alpha_i = firm fixed effects
   - delta_t = year fixed effects
   - SE clustered at firm level
   - DV at t+1 (forward-looking)
  </action>
