**Key Implementation Requirements:**

1. **Load All Components:**

   - Load all three parquet files
   - Verify gvkey is zero-padded to 6 characters (string format)
   - Verify fyear is integer type

2. **Merge into FINAL_PANEL:**
   - Start with abnormal_investment as base (has DV + controls)
   - Merge PRiskFY on (gvkey, fyear) - left join
   - Merge style_frozen on (gvkey, fyear) - left join
   - Result: one row per (gvkey, fyear) with all variables

3. **Sample Filters (LOCKED):**
   - DROP if AbsAbInv is missing
   - DROP if PRiskFY is missing
   - DROP if style_frozen is missing
   - DROP if any control missing (ln_at_t, lev_t, cash_t, roa_t, mb_t, SalesGrowth)
   - DROP if fyear is first year for gvkey (already handled in 58-03)

4. **Create Interaction Term:**
   - interact = PRiskFY * style_frozen
   - This is the MODERATOR variable
   - Tests whether PRisk effect varies with CEO style

5. **Set Up Panel for Regression:**
   - Set MultiIndex: (gvkey as entity, fyear as time)
   - Use linearmodels.PanelOLS for Firm FE + Year FE
   - Alternative: statsmodels OLS with entity dummies (if PanelOLS unavailable)

6. **H9 Regression Model:**
   ```
   AbsAbInv_{i,t+1} = β0 + β1*PRiskFY_t + β2*StyleFrozen_t
                    + β3*(PRiskFY_t × StyleFrozen_t)
                    + γ1*ln_at_t + γ2*lev_t + γ3*cash_t + γ4*roa_t + γ5*mb_t + γ6*SalesGrowth_t
                    + FirmFE_i + YearFE_t + ε
   ```

   **Fixed Effects:**
   - Firm FE: One dummy per gvkey
   - Year FE: One dummy per fyear

   **Standard Errors:**
   - Cluster by gvkey (firm level)
   - Use linearmodels: cov_type='clustered', cluster_entity=True

7. **Coefficient of Interest: β3 (Interaction)**
   - If β3 > 0: PRisk associated with LARGER abnormal investment under vaguer CEOs
   - If β3 ≈ 0: CEO vagueness does NOT moderate PRisk → abnormal investment (meaningful null)
   - If β3 < 0: Vagueness dampens abnormal investment response to PRisk

8. **Sanity Checks (MANDATORY):**
   Before running regression, print and save:

   a. **PRiskFY Coverage:**
      - mean, SD, p1, p99 of PRiskFY
      - min/max fiscal years present
      - N_obs with PRiskFY

   b. **StyleFrozen Coverage:**
      - mean, SD, p1, p99 of style_frozen
      - Confirm NOT constant within CEO (though time-invariant by construction)
      - Variance by firm (CEOs may move)

   c. **DV and Controls:**
      - mean, SD, min, max of AbsAbInv
      - Control variable distributions
      - Winsorization confirmation (p1, p99 at expected bounds)

   d. **Biddle Cell Viability:**
      - Fraction of (ind2, fyear) cells with N >= 30
      - Fraction of observations dropped due to small cells

9. **Regression Output:**
   Save to multiple formats:
   - h9_regression_results.csv: Coefficient, std_error, t_stat, p_value for each variable
   - h9_regression_output.txt: Full model summary (R2, N, N_firms, FE stats)
   - Summary report with interpretation

10. **Output Files:**
    - final_panel.parquet: Merged dataset ready for analysis
    - h9_regression_results.csv: Regression coefficients
    - h9_regression_output.txt: Full output
    - report_step58_04.md: Summary with interpretation
