<objective>
Construct Biddle-style abnormal investment (AbsAbInv) and control variables for H9 regression.

AbsAbInv_{i,t+1} measures the absolute deviation of firm i's actual investment in year t+1 from expected investment given its sales growth in year t. Controls are measured at year t to predict investment in t+1.

Purpose: Implement Biddle et al. (2009) investment efficiency measure as the dependent variable for H9 regression, along with all required control variables.

Output: abnormal_investment.parquet with columns (gvkey, fyear, AbsAbInv, TotalInv, SalesGrowth, ln_at_t, lev_t, cash_t, roa_t, mb_t).
</objective>

Create new script 2_Scripts/5_Financial_V3/5.8_H9_AbnormalInvestment.py implementing Biddle et al. (2009) abnormal investment measure.

**Input File:**
Compustat: 1_Inputs/comp_na_daily_all/comp_na_daily_all.parquet
- Required columns: gvkey, datadate, fyear, sic
- Investment components: capx, xrd, aqc, sppe, at
- Sales: sale
- Controls: oibdp (or ni), dltt, dlc, che, prcc_f, csho

**Key Implementation Requirements:**

1. **Load Compustat Data:**
   - Read all required columns from Compustat parquet
   - Zero-pad gvkey to 6 characters
   - Convert datadate to datetime
   - Filter to 2002 <= fyear <= 2018 (sample period)

2. **Create Panel with Leads/Lags:**
   For each gvkey, sort by fyear:
   - at_t = at (current year assets, for scaling)
   - sale_t = sale (current year sales)
   - sale_tm1 = lag(sale, 1) (prior year sales, for growth)
   - capx_tp1 = lead(capx, 1) (next year CAPEX)
   - xrd_tp1 = lead(xrd, 1) (next year R&D)
   - aqc_tp1 = lead(aqc, 1) (next year acquisitions)
   - sppe_tp1 = lead(sppe, 1) (next year asset sales)

3. **Apply Investment Missingness Rules (NON-NEGOTIABLE):**
   For each firm-year t:
   - DROP if at_t is missing OR at_t <= 0
   - DROP if capx_tp1 is missing (CAPX required)
   - Set xrd_tp1 = 0 if missing (R&D optional)
   - Set aqc_tp1 = 0 if missing (acquisitions optional)
   - Set sppe_tp1 = 0 if missing (asset sales optional)

4. **Construct Total Investment at t+1:**
   TotalInv_{t+1} = (capx_tp1 + xrd_tp1 + aqc_tp1 - sppe_tp1) / at_t
   - Denominator is at_t (year t assets, not t+1)
   - This is "investment in t+1 scaled by assets at t"

5. **Construct Sales Growth at t:**
   SalesGrowth_t = (sale_t - sale_tm1) / sale_tm1
   - DROP if sale_tm1 is missing OR sale_tm1 <= 0
   - DROP if sale_t is missing
   - This creates the predictor for expected investment

6. **Industry Classification:**
   - ind2 = floor(sic / 100) (two-digit SIC)
   - ind1 = floor(sic / 1000) (one-digit SIC, fallback)
   - DROP if sic is missing
   - DROP if sic < 100 OR sic > 9999
   - DROP utilities: 4900 <= sic <= 4999
   - DROP financials: 6000 <= sic <= 6999

7. **First-Stage Expected Investment Model:**
   For each (ind2, fyear=t) cell with N >= 30:
     Run OLS: TotalInv_{i,t+1} = a + b*SalesGrowth_{i,t} + e

   **Cell Size Rule (LOCKED):**
   - If N < 30 for (ind2, fyear) cell:
     - Try fallback to (ind1, fyear) with N >= 30
     - If still N < 30, DROP those observations for that year
   - Report: fraction of cells with N >= 30, fraction dropped

8. **Compute Abnormal Investment:**
   - AbInv_{t+1} = residual e from first-stage
   - AbsAbInv_{t+1} = abs(AbInv_{t+1})
   - This is the DEPENDENT VARIABLE for H9

9. **Construct Controls at Year t:**
   All controls measured at t to predict DV at t+1:
   - **Size:** ln_at_t = ln(at_t)
   - **Leverage:** lev_t = (dltt + dlc) / at_t
     - If dltt or dlc missing, treat as 0
     - Require at_t > 0
   - **Cash:** cash_t = che / at_t
     - If che missing, set to 0
   - **Profitability (ROA):**
     - Preferred: roa_t = oibdp / at_t
     - If oibdp missing but ni exists: roa_t = ni / at_t
     - If both missing: roa_t = missing, DROP later
   - **Market-to-book:**
     - mve_t = prcc_f * csho (market equity)
     - Require both non-missing and mve_t > 0
     - mb_t = mve_t / at_t
     - If fails, mb_t = missing, DROP later
   - **SalesGrowth_t:** Already computed above

10. **Winsorization (LOCKED RULE):**
    Winsorize at 1st and 99th percentiles BY fyear:
    - Variables to winsorize: AbsAbInv, TotalInv, SalesGrowth, ln_at_t, lev_t, cash_t, roa_t, mb_t
    - For each fyear, compute p1 and p99 from non-missing values
    - Replace values < p1 with p1, values > p99 with p99
    - Use pandas .clip() or custom function

11. **Output Dataset:**
    - Columns: gvkey, fyear, AbsAbInv, TotalInv, SalesGrowth, ln_at_t, lev_t, cash_t, roa_t, mb_t, ind2, sic
    - One row per firm-year t (where t+1 investment exists)
    - All variables winsorized by fyear
    - First-year observations dropped (no SalesGrowth)

**Script Structure:**
- Imports: sys, os, pathlib, pandas, numpy, argparse, statsmodels.api
- CLI: --dry-run, --min-cell-size (default 30)
- load_compustat(): Read and clean Compustat data
- create_leads_lags(): Build tp1 and tm1 variables
- construct_investment(): Build TotalInv with missingness rules
- construct_sales_growth(): Build SalesGrowth with missingness rules
- assign_industry(): Create ind1, ind2, exclude utilities/financials
- run_first_stage(): OLS by (ind2, fyear) with ind1 fallback
- construct_controls(): Build all control variables
- winsorize_by_fyear(): Apply 1%/99% winsorization
- generate_report(): Investment stats, first-stage diagnostics, control distributions
- main(): Orchestrate execution

**Anti-Patterns to Avoid:**
- DO NOT set missing CAPX to 0 (CAPX is required)
- DO NOT use at_tp1 as denominator (use at_t)
- DO NOT winsorize before first-stage regression (winsorize after)
- DO NOT use firm fixed effects in first-stage (use industry-year cells)
- DO NOT drop observations if R&D/AQC/SPPE missing (set to 0)
- DO NOT include first-year observations (no lagged sales)
