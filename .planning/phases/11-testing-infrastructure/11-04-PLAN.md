    Example:

    ```python
    #!/usr/bin/env python3
    """Generate baseline checksums for regression testing."""

    import hashlib
    import json
    import pandas as pd
    from pathlib import Path


    def compute_dataframe_checksum(df: pd.DataFrame) -> str:
        """Compute SHA-256 checksum of a DataFrame."""
        return hashlib.sha256(
            pd.util.hash_pandas_object(df, index=False).values.tobytes()
        ).hexdigest()


    def compute_file_checksum(filepath: Path) -> str:
        """Compute SHA-256 checksum of a Parquet file."""
        df = pd.read_parquet(filepath)
        return compute_dataframe_checksum(df)


    def generate_baseline_checksums():
        """Generate baseline checksums from existing outputs."""
        # Define key output files to track
        key_outputs = {
            "step1_cleaned_metadata": "4_Outputs/1.1_CleanMetadata/latest/cleaned_metadata.parquet",
            "step3_financial_features": "4_Outputs/3_Financial_Features/3.0_BuildFinancialFeatures/latest/financial_features.parquet",
        }

        # Add Step 2 yearly outputs
        for year in range(2002, 2019):
            key_outputs[f"step2_linguistic_counts_{year}"] = (
                f"4_Outputs/2_Textual_Analysis/2.1_Tokenized/latest/linguistic_counts_{year}.parquet"
            )

        # Compute checksums
        baseline_checksums = {}
        missing_files = []

        for key, file_path in key_outputs.items():
            path = Path(file_path)
            if path.exists():
                checksum = compute_file_checksum(path)
                baseline_checksums[key] = checksum
                print(f"[OK] {key}: {checksum[:32]}...")
            else:
                print(f"[SKIP] {key}: File not found ({file_path})")
                missing_files.append(key)

        # Add metadata
        baseline_checksums["_metadata"] = {
            "generated_at": pd.Timestamp.now().isoformat(),
            "git_commit": None,  # Can be populated by subprocess
            "total_files": len([k for k in key_outputs.keys() if k not in missing_files]),
            "missing_files": missing_files,
        }

        # Save to JSON
        baseline_path = Path("tests/fixtures/baseline_checksums.json")
        baseline_path.parent.mkdir(parents=True, exist_ok=True)

        with open(baseline_path, "w") as f:
            json.dump(baseline_checksums, f, indent=2)

        print(f"\nBaseline checksums saved to: {baseline_path}")
        print(f"Total files: {len([k for k in baseline_checksums.keys() if not k.startswith('_')])}")
        print(f"Missing files: {len(missing_files)}")


    if __name__ == "__main__":
        generate_baseline_checksums()
    ```

    Note: This script is NOT a test - it's a helper for generating baselines.

  <action>
    Verify regression tests detect changes by intentionally modifying a small part of an output file and checking if the test fails.

    Note: This is a one-time verification step - DO NOT commit the modified file.

    Steps:
    1. Copy an output file to a temporary location
    2. Modify a single value in the temporary file
    3. Compute checksum of modified file
    4. Verify checksum differs from baseline
    5. Clean up (delete temporary file)

    Example verification (can be run as a separate script):

    ```python
    #!/usr/bin/env python3
    """Verify regression tests detect changes."""

    import pandas as pd
    from pathlib import Path

    # Load original file
    original_file = Path("4_Outputs/1.1_CleanMetadata/latest/cleaned_metadata.parquet")
    df_original = pd.read_parquet(original_file)

    # Modify first row
    df_modified = df_original.copy()
    if len(df_modified) > 0:
        df_modified.iloc[0, 0] = "MODIFIED_FOR_TEST"

    # Compute checksums
    from hashlib import sha256

    checksum_original = sha256(
        pd.util.hash_pandas_object(df_original, index=False).values.tobytes()
    ).hexdigest()

    checksum_modified = sha256(
        pd.util.hash_pandas_object(df_modified, index=False).values.tobytes()
    ).hexdigest()

    print(f"Original checksum: {checksum_original}")
    print(f"Modified checksum: {checksum_modified}")
    print(f"Checksums differ: {checksum_original != checksum_modified}")

    assert checksum_original != checksum_modified, "Checksums should differ!"
    print("✓ Regression detection verified - checksums differ when data changes")
    ```

    Run the verification script once to confirm checksum comparison works, then delete it.

    ```markdown
    # Regression Tests

    ## Purpose

    Regression tests ensure that code changes don't silently alter output data. They compare SHA-256 checksums of output files against baseline checksums.

    ## Running Regression Tests

    ```bash
    # Run all regression tests
    pytest tests/regression/ -v -m regression

    # Run specific regression test
    pytest tests/regression/test_output_stability.py::test_regression_step1_output_stability -v
    ```

    ## Updating Baselines

    When code changes intentionally alter outputs, update the baseline checksums:

    1. Run the pipeline scripts to generate new outputs
    2. Generate new baseline checksums:
       ```bash
       python tests/regression/generate_baseline_checksums.py
       ```
    3. Verify the changes are intentional:
       ```bash
       git diff tests/fixtures/baseline_checksums.json
       ```
    4. Commit the updated baseline:
       ```bash
       git add tests/fixtures/baseline_checksums.json
       git commit -m "test(regression): update baseline for [reason]"
       ```

    ## Missing Baselines

    If a regression test skips with "No baseline checksum", it means the baseline checksums haven't been generated yet. Run `python tests/regression/generate_baseline_checksums.py` to create them.

    ## Intentional Regressions

    If a regression test fails but the change is intentional:
    1. Review the diff in checksums
    2. Verify the output changes are expected
    3. Update the baseline checksums (see above)
    4. Commit the changes with a clear message explaining the reason

    ## Checksum Algorithm

    Regression tests use SHA-256 checksums computed via pandas:
    ```python
    hashlib.sha256(pd.util.hash_pandas_object(df, index=False).values.tobytes()).hexdigest()
    ```

    This ensures bitwise-identical detection of data changes.
