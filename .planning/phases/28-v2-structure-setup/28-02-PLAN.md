# Known Pitfalls from STATE.md
1. **FE Collinearity Trap:** Firm + industry FE are redundant (firms don't change industries). Use `drop_absorbed=False, check_rank=True`
2. **Interaction Multicollinearity:** Always center continuous variables before creating interactions
3. **Weak Instruments in 2SLS:** F < 10 on first stage means 2SLS is more biased than OLS. Enforce F > 10 programmatically
4. **Manager FE Connectivity:** With few CEO "movers" between firms, CEO FE conflates manager style with firm culture
5. **Multiple Testing in Subsamples:** Use interaction terms instead of separate regressions to avoid false positives

# Econometric Infrastructure Requirements (ECON-01 to ECON-07)
- Panel OLS with firm + year + industry fixed effects (Fama-French 48)
- Interaction term creation with mean-centering
- Clustered standard errors (firm-level with double-clustering option)
- 2SLS with instruments: manager's prior-firm vagueness, industry-peer average vagueness
- Instrument validation: first-stage F > 10, Hansen J overidentification test
- Newey-West adjustment for heteroskedasticity and autocorrelation
- VIF < 5 multicollinearity diagnostics

# Regression Specifications (from REQUIREMENTS.md)

## H1: Cash Holdings
- Model: CashHoldings_{t+1} = β0 + β1*Uncertainty_t + β2*Leverage_t + β3*(Uncertainty×Leverage) + Controls + Firm_FE + Year_FE + Industry_FE + ε
- Expected: β1 > 0 (vagueness increases cash), β3 < 0 (leverage attenuates)
- Standard errors clustered at firm level

## H2: Investment Efficiency
- Model: Efficiency_{t+1} = β0 + β1*Uncertainty_t + β2*Leverage_t + β3*(Uncertainty×Leverage) + Controls + FEs + ε
- Expected: β1 < 0 (vagueness lowers efficiency), β3 > 0 (leverage improves)
- Test both Efficiency Score and ROA Residual DVs

## H3: Payout Policy
- Stability Model: Stability_{t+1} = β0 + β1*Uncertainty_t + β2*Leverage_t + β3*(Uncertainty×Leverage) + Controls + FEs + ε
  - Expected: β1 < 0, β3 < 0
- Flexibility Model: Flexibility_{t+1} = β0 + β1*Uncertainty_t + β2*Leverage_t + β3*(Uncertainty×Leverage) + Controls + FEs + ε
  - Expected: β1 > 0, β3 > 0
