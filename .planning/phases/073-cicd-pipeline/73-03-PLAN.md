<task type="auto">
  <name>Task 1: Create pre-commit configuration per TOOL-02</name>
  <files>.pre-commit-config.yaml</files>
  <action>
    Create `.pre-commit-config.yaml` following DOC_TOOLING_STANDARD.md TOOL-02:

    ```yaml
    # Pre-commit hooks configuration
    # See https://pre-commit.com for more information
    # See https://pre-commit.com/hooks.html for more hooks
    #
    # Installation:
    #   pip install pre-commit
    #   pre-commit install
    #   pre-commit run --all-files  # First-time run

    repos:
      # General file quality
      - repo: https://github.com/pre-commit/pre-commit-hooks
        rev: v5.0.0
        hooks:
          # Whitespace and line endings
          - id: trailing-whitespace
          - id: end-of-file-fixer

          # Config file validation
          - id: check-yaml
            args: [--unsafe]
          - id: check-toml
          - id: check-json

          # File safety
          - id: check-added-large-files
            args: ['--maxkb=1000']
          - id: check-merge-conflict
          - id: check-case-conflict

          # Security
          - id: detect-private-key
          - id: debug-statements

      # Ruff - linter and formatter
      - repo: https://github.com/astral-sh/ruff-pre-commit
        rev: v0.9.0
        hooks:
          # Linter
          - id: ruff
            args: [--fix, --exit-non-zero-on-fix]
          # Formatter
          - id: ruff-format

      # Type checking with mypy
      - repo: https://github.com/pre-commit/mirrors-mypy
        rev: v1.14.1
        hooks:
          - id: mypy
            additional_dependencies:
              - pydantic>=2.0
              - types-PyYAML
            args: [--config-file=pyproject.toml]
            pass_filenames: false
            entry: mypy src/f1d/shared

    # CI configuration (for pre-commit.ci if used)
    ci:
      autofix_commit_msg: |
        [pre-commit.ci] auto fixes from pre-commit.com hooks

        for more information, see https://pre-commit.ci
      autofix_prs: true
      autoupdate_branch: ''
      autoupdate_commit_msg: '[pre-commit.ci] pre-commit autoupdate'
      autoupdate_schedule: weekly
      skip: []
      submodules: false
    ```

    Key points:
    - Uses pre-commit-hooks v5.0.0 (latest stable)
    - Uses ruff-pre-commit v0.9.0 matching pyproject.toml version
    - Uses mypy v1.14.1 matching pyproject.toml version
    - mypy runs only on Tier 1 modules (src/f1d/shared) to match CI
    - Includes pydantic and types-PyYAML for mypy
    - File quality hooks prevent common issues
    - Large file check at 1MB threshold
  </action>
  <verify>
    test -f .pre-commit-config.yaml && echo "pre-commit config created"
    grep -E "ruff-pre-commit" .pre-commit-config.yaml && echo "ruff hook present"
    grep -E "mirrors-mypy" .pre-commit-config.yaml && echo "mypy hook present"
  </verify>
  <done>

  </done>
