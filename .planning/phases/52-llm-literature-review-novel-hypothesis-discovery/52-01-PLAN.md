  <files>.planning/phases/52-llm-literature-review-novel-hypothesis-discovery/52-01-LITERATURE-MATRIX.md</files>
  <action>
Create a comprehensive literature evidence matrix by:

1. **Paper Catalogue (2023-2026):**
   - Start with papers from 52-RESEARCH.md (Kelly et al., Kim et al., Lopez-Lira & Tang)
   - Use Google Scholar search for "LLM earnings call" OR "GPT financial disclosure" OR "large language model textual analysis finance" (2023-2026)
   - Search SSRN for recent working papers on LLM + corporate disclosure
   - Include all papers that use LLMs/GPT/BERT for earnings call analysis
   - Target: 20-30 papers minimum

2. **Extract from Each Paper:**
   - Independent Variable (IV) used
   - Dependent Variable (DV) tested
   - Sample size
   - Main finding (significant/null)
   - Text measure methodology (dictionary, embedding, LLM classification, etc.)

3. **Create Evidence Matrix:**
   - Rows: All DVs tested in literature (returns, volatility, dispersion, turnover, M&A, compensation, etc.)
   - Columns: All IVs tested (sentiment, uncertainty, evasiveness, complexity, FLS specificity, etc.)
   - Cells: Paper citations + finding (significant/null)
   - Highlight EMPTY CELLS = true literature gaps

4. **Gap Analysis:**
   - List all IV-DV combinations with NO prior testing
   - List combinations with only 1-2 prior tests (replication opportunities)
   - Identify methodological gaps (e.g., within-firm vs cross-sectional)

Format document with:
- Metadata section (search terms, date, sources)
- Paper catalogue table (author, year, title, journal, key finding)
- Evidence matrix (visual grid)
- Gap analysis summary
  </action>
  <verify>Document contains 20+ papers catalogued, evidence matrix has at least 8 DVs x 6 IVs, gap analysis section identifies at least 10 untested IV-DV combinations</verify>
  <done>Literature evidence matrix complete with clear identification of true literature gaps suitable for novel hypothesis generation</done>
