  <files>.planning/phases/52-llm-literature-review-novel-hypothesis-discovery/52-04-RED-TEAM-VERIFICATION.md</files>
  <action>
Execute Red Team adversarial verification from 52-RESEARCH.md methodology:

**For Each Candidate Hypothesis (prioritized by preliminary score):**

**Challenge 1: Literature Gap Challenge**
```
Question: "Has this EXACT hypothesis been tested before?"
Search: Google Scholar, SSRN for exact IV-DV combination
```

| Criterion | Kill Threshold | Evidence Required |
|-----------|----------------|-------------------|
| Direct prior test | >500 observations | Paper citation, sample size |
| Partial prior test | Related but different | Document how this differs |
| No prior test | SURVIVES | Document search terms used |

**Challenge 2: Data Feasibility Challenge**
```
Question: "Can we measure BOTH IV and DV with available data?"
Source: 52-02-FEASIBILITY-MATRIX.md
```

| Criterion | Kill Threshold | Evidence Required |
|-----------|----------------|-------------------|
| Observations | <5,000 after merges | Verified N from 52-02 |
| Temporal overlap | <5 years overlap | Coverage analysis |
| Key variable missing | Cannot construct | Specific gap identified |

**Challenge 3: Power/Effect Size Challenge**
```
Question: "Given available N, can we detect meaningful effect?"
Source: 52-02-POWER-CALCULATIONS.md
```

| Criterion | Kill Threshold | Evidence Required |
|-----------|----------------|-------------------|
| Power for small effect | <80% | Power calculation |
| Within-firm variation | ICC >0.7 (90%+ between-firm) | ICC estimate |
| Panel FE feasible | <1,000 unique firms | Firm count |

**Challenge 4: Mechanism Challenge**
```
Question: "What's the theoretical channel connecting IV to DV?"
```

| Criterion | Kill Threshold | Evidence Required |
|-----------|----------------|-------------------|
| Causal story | No clear channel | Mechanism description |
| Obvious confounders | Uncontrollable | Confounder list |
| Reverse causality | DV→IV more plausible | Direction argument |

**Documentation Format for Each Hypothesis:**

```markdown
## HXX: [Name]

### Challenge 1: Literature Gap
- Search terms: [...]
- Results: [PASS/FAIL]
- Evidence: [paper or "no prior test found"]

### Challenge 2: Data Feasibility  
- Estimated N: [number]
- Temporal overlap: [years]
- Missing variables: [none/list]
- Result: [PASS/FAIL]

### Challenge 3: Power
- Power for f2=0.02: [%]
- Within-firm variance: [estimate]
- Result: [PASS/FAIL]

### Challenge 4: Mechanism
- Causal story: [plausible/weak/none]
- Confounders: [controlled/uncontrolled]
- Direction: [clear/ambiguous]
- Result: [PASS/FAIL]

### Verdict: [SURVIVES / KILLED]
- Reason: [if killed, specific failure point]
```
