  <action>
    Critical lesson from H1-H6: Fixed effects absorb between-firm variation. Compute within-firm variance for key candidate measures:
    
    1. **Load existing text measures from Step 2 outputs:**
       - Reference: 4_Outputs/2_Text/latest/ or timestamped folder
       - Variables: Manager_QA_Uncertainty_pct, Manager_QA_Weak_Modal_pct, etc.
       - Available: 1,785 text measure variables (15 roles x 8 categories x 3 contexts)
    
    2. **For each candidate measure, compute:**
       - Overall variance (σ²_total)
       - Between-firm variance (σ²_between)
       - Within-firm variance (σ²_within)
       - Intraclass correlation (ICC = σ²_between / σ²_total)
       
       Formula: Within-firm % = 1 - ICC
    
    3. **Target threshold:**
       - Within-firm variance > 30% is GOOD (enough variation for Firm FE)
       - Within-firm variance 20-30% is MARGINAL
       - Within-firm variance < 20% is POOR (FE will absorb most variation)
    
    4. **Key measures to analyze:**
       - Uncertainty_pct (baseline)
       - Weak_Modal_pct (hedging)
       - QA vs Pres gap (spontaneous vs scripted)
       - Forward-looking statements (to be computed)
    
    5. **For FirmLevelRisk measures:**
       - PRisk, NPRisk have established within-firm variance (Hassan verified)
       - Compute for topic-specific risks (PRiskT_*)
    
    Create table: Measure | σ²_total | σ²_within | Within % | Assessment
  </action>
  <verify>
    Within-firm variance section contains:
    - At least 10 measures analyzed
    - Each has: total variance, within variance, within %
    - Assessment (GOOD/MARGINAL/POOR) for each

  </verify>
