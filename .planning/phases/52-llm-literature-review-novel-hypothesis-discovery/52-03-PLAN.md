Generate candidate hypotheses using Blue Team methodology from 52-RESEARCH.md:

**Generation Sources:**

1. **From Literature Gaps (52-01-LITERATURE-MATRIX.md):**
   - Identify EMPTY cells in evidence matrix
   - Each empty cell = potential hypothesis
   - Prioritize cells where adjacent cells show significant effects

2. **From LLM Capabilities (52-01-LLM-CAPABILITY-INVENTORY.md):**
   - Which capabilities have NOT been applied to available DVs?
   - Cross-reference capability list with untested DVs

3. **From H1-H6 Null Results Analysis:**
   - WHY did dictionary measures fail?
   - What DIFFERENT measures might work?
   - Consider: dynamics (change over time), quality (not counts), context (Q&A vs Pres)

4. **From High-Priority Candidates in 52-RESEARCH.md:**
   - CEO Q&A Evasiveness → Returns
   - Narrative Inconsistency → Volatility
   - FLS Specificity → Analyst Accuracy
   - CEO-CFO Alignment → M&A Premium
   - Information Novelty → Volume/Price Discovery
   - LM Complexity → Investment Efficiency
   - Question Re-asking → Bad News

**For Each Candidate Hypothesis, Document:**

```markdown
## Hypothesis HXX: [Name]

**Prediction:** [IV] → [expected sign] [DV]

**IV Definition:**
- Measure: [specific text measure]
- Methodology: [dictionary/LLM/embedding]
- Level: [call/quarter/year]

**DV Definition:**
- Outcome: [specific financial outcome]
- Source: [CRSP/IBES/Execucomp/etc.]
- Construction: [how computed]

**Theoretical Mechanism:**
[Why should IV affect DV? What is the causal story?]

**Novelty Claim:**
- Closest prior test: [paper or "none found"]
- How this differs: [specific difference]

**Preliminary Feasibility:**
- Data available: [yes/partial/no]
- Estimated N: [from 52-02]
- Power: [adequate/excellent]

**Preliminary Score:**
- Novelty (0-1): [score]
- Feasibility (0-1): [score]  
- Confidence (0-1): [score]
- **Overall (weighted):** [0.35*Nov + 0.35*Feas + 0.30*Conf]
```

5. **Target:** 20-30 candidates with full documentation

6. **Rank by Overall Score** (descending)

7. **Flag for Red Team:**
   - Candidates scoring ≥0.85: "Priority verification"
   - Candidates scoring 0.70-0.84: "Standard verification"
   - Candidates scoring <0.70: "Low priority"
