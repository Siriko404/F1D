  <action>
    For each candidate, document full specification following 5E Rule:
    
    **5E Rule Checklist:**
    1. **Explicit**: Clearly stated IV → DV with expected sign
    2. **Evidence-based**: Rooted in prior literature or theory
    3. **Ex-ante**: Formulated BEFORE testing (pre-registration quality)
    4. **Explanatory**: Clear "why" mechanism
    5. **Empirically Testable**: Operationalizable with available data
    
    **For each candidate, document:**
    
    ```markdown
    ### Candidate H{N}: {Title}
    
    **Hypothesis Statement:**
    {IV} → {DV} (expected sign: +/-)
    
    **Theoretical Mechanism:**
    {2-3 sentences explaining WHY this relationship should exist}
    
    **Operationalization:**
    - IV: {exact measure, source dataset}
    - DV: {exact measure, source dataset}
    - Controls: {list of controls}
    - FE: {fixed effects specification}
    
    **Sample Estimate:**
    - From 52-02: {N observations, source}
    - Power: {HIGH/MEDIUM/LOW}
    
    **Novelty Claim:**
    - Gap from 52-01: {which gap this fills}
    - Why not tested before: {explanation}
    
    **Tier:** {1/2/3/4}
    ```
  </action>
  <verify>

    - Each hypothesis has explicit sign prediction
    - Mechanism documented for each
    - Sample size referenced from 52-02
