## Key Decisions

1. **[77-01]** Stage 2 text scripts migrated to `src/f1d/text/` with `f1d.shared.*` imports
2. **[77-02]** Dynamic imports eliminated - `1.5_Utils.py` moved to `src/f1d/shared/sample_utils.py`
3. **[77-03]** Used cause-specific Cox hazards instead of FineGrayAFTFitter (not available in lifelines 0.30.0)
4. **[77-04]** Created separate test file per hypothesis for maintainability (H1-H4, H5-H9)
5. **[77-05]** Used subprocess.run() for script isolation in dry-run tests
6. **[77-07]** Used typing.cast instead of type: ignore for pandas type inference issues
7. **[77-09]** Used pandas-stubs instead of types-pandas (modern official package)
8. **[77-11]** Documented type ignores with TYPE ERROR BASELINE pattern instead of fixing
