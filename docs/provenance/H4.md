## A. Suite Overview

### A1) Identity

| Field | Value |
|-------|-------|
| **Hypothesis Suite Name/ID** | H4 Leverage Discipline |
| **Research Question** | Does lagged leverage discipline manager speech? Tests whether higher prior-year leverage reduces current uncertainty in managerial communication. |
| **Estimation Unit** | Call-level (individual earnings call) |
| **Primary Keys** | `file_name` (unique call identifier), `gvkey` (firm identifier), `year` (call year) |
| **Panel Type** | Unbalanced panel with firm + year fixed effects |

### A2) Model Family and Estimator

| Field | Value |
|-------|-------|
| **Model Family** | Panel Fixed Effects OLS |
| **Estimator** | `linearmodels.panel.PanelOLS` with within transformation |
| **Objective/Link** | Linear regression (OLS) |
| **Within Transformation** | Entity (firm) + Time (year) effects absorbed via `EntityEffects + TimeEffects` |
| **Drop Absorbed** | `drop_absorbed=True` removes collinear controls |
| **Weighting** | None (unweighted) |

### A3) Outcome(s) / Target Definition

| Variable | Description | Coding Rules | Timing |
|----------|-------------|--------------|--------|
| `Manager_QA_Uncertainty_pct` | Manager Q&A uncertainty % (all managers incl. CEO) | LM Uncertainty words / total words × 100; NaN if no Q&A speech | t (current call) |
| `CEO_QA_Uncertainty_pct` | CEO-only Q&A uncertainty % | LM Uncertainty words / total words × 100; NaN if CEO has no Q&A speech | t |
| `Manager_QA_Weak_Modal_pct` | Manager Q&A weak modal % | LM Weak_Modal words / total words × 100 | t |
| `CEO_QA_Weak_Modal_pct` | CEO-only Q&A weak modal % | LM Weak_Modal words / total words × 100 | t |
| `Manager_Pres_Uncertainty_pct` | Manager Presentation uncertainty % | LM Uncertainty words / total words × 100; NaN if no pres speech | t |
| `CEO_Pres_Uncertainty_pct` | CEO-only Presentation uncertainty % | LM Uncertainty words / total words × 100 | t |

**Missingness Policy:** Listwise deletion per regression — rows with NaN in DV, `Lev_lag`, or controls are dropped via `dropna(subset=required)`.

### A4) Main RHS and Controls

**Key Independent Variable:**
| Variable | Description |
|----------|-------------|
| `Lev_lag` | One-year lagged leverage (Lev at t-1), computed per firm per fiscal year |

**Dynamic Covariate (added based on DV):**
- If DV is a QA measure → corresponding Pres **Uncertainty** measure added as control
- E.g., `Manager_QA_Uncertainty_pct` regressions include `Manager_Pres_Uncertainty_pct`
- Note: Weak_Modal QA DVs also use Pres Uncertainty (not Pres Weak_Modal) as control — the primary uncertainty measure is used for all speaker roles

**Base Controls (all specs):**
| Variable | Description |
|----------|-------------|
| `Analyst_QA_Uncertainty_pct` | Analyst question uncertainty (controls for call difficulty) |
| `Size` | Log total assets (ln(atq)) |
| `TobinsQ` | Tobin's Q (market-to-book ratio) |
| `ROA` | Return on assets |
| `CashHoldings` | Cash / total assets |
| `DividendPayer` | Binary: 1 if firm paid dividends in fiscal year |
| `firm_maturity` | Retained earnings / total assets (req / atq) |
| `earnings_volatility` | Rolling 5-year SD of quarterly ROA |

**Sample Filters:**
| Filter | Definition |
|--------|------------|
| `min_calls >= 5` | Firms must have ≥ 5 calls to be included in regression |
| Industry samples | Main (FF12 ≠ 8,11), Finance (FF12 = 11), Utility (FF12 = 8) |

### A5) Fixed Effects & Variance Estimator

| Component | Specification |
|-----------|---------------|
| **Entity Effects** | `C(gvkey)` — firm fixed effects via `EntityEffects` |
| **Time Effects** | `C(year)` — year fixed effects via `TimeEffects` |
| **Standard Errors** | Clustered by firm (`cov_type="clustered", cluster_entity=True`) |
| **DF Adjustment** | Default (linearmodels handles) |

### A6) Model-Family Addendum

**N/A** — Standard panel OLS, no IV, survival, DiD, or GLM components.

## J. Known Issues / Ambiguities (Actionable)

### Issue 1: CEO Identification Missingness

**Description:** `CEO_QA_Uncertainty_pct` has 32% missingness vs 4.2% for `Manager_QA_Uncertainty_pct`.

**Impact:** Smaller sample size for CEO-specific regressions (N ≈ 55K vs 76K).

**Root Cause:** CEO identification requires speaker role classification; not all calls have CEO identified.

**Recommended Fix:** None required — missingness is by design (CEO not always identifiable). Document in paper.

### Issue 2: Lev_lag Requires Consecutive Fiscal Years

**Description:** `Lev_lag` set to NaN if fiscal years are non-consecutive (gap in Compustat coverage).

**Impact:** 6.7% of calls have missing `Lev_lag`.

**Code Location:** `build_h4_leverage_panel.py:104-105`

```python
is_consecutive = (firm_year["fyearq_int"] - firm_year["prev_fyearq"]) == 1
firm_year.loc[~is_consecutive, "Lev_lag"] = np.nan
```

**Recommended Fix:** None — this is correct behavior. Alternative would introduce measurement error.

### Issue 3: Within-R² Reporting (RESOLVED)

**Description:** LaTeX table originally reported inflated within-R² values (0.63–0.92) due to incorrect one-step additive demeaning instead of proper two-way FE within transformation.

**Fix:** `run_h4_leverage.py` now uses `model.rsquared_within` directly from linearmodels, stored as both `rsquared` and `within_r2` in meta dict. LaTeX table reports the correct PanelOLS within-R² (0.0002–0.027).

**Status:** RESOLVED — correct within-R² values now reported in all artifacts.

### Issue 4: Lev_lag Timing Convention

**Description:** Lev_lag uses fiscal year (fyearq) not calendar year for lagging.

**Implication:** For firms with non-calendar fiscal years, Lev_lag may not align with calendar year t-1.

**Status:** By design — fiscal year alignment is standard in accounting research.
