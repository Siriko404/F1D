## B. Redesign Rationale

The old H4 tested whether **leverage disciplines speech** (Lev_lag → uncertainty measures). The redesign flips the direction: **does speech uncertainty predict leverage?** (uncertainty → Lev).

This follows the H1 pattern exactly:
- 6 simultaneous IVs (same uncertainty/clarity measures)
- 2 DVs (contemporaneous + lead)
- Industry/Firm FE crossed with Base/Extended controls
- 8-column LaTeX table
- Main sample only
