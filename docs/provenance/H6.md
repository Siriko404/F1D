## G. Outliers, Missing Data, Scaling, and Model-Specific Data Rules

### Winsorization

| Engine | Method | Variables | Thresholds | When Applied |
|--------|--------|-----------|------------|--------------|
| **CompustatEngine** | Per-year | Size, Lev, ROA, CashHoldings, TobinsQ, etc. | 1%/99% per-year | At engine load time |
| **LinguisticEngine** | Per-year | All `_pct` columns (105) | 0%/99% per-year (lower=0.0) | At engine load time |
| **CCCL Instrument** | None | `shift_intensity_*` | N/A | Pre-computed externally |

### Missing Data Policy

| Scenario | Handling | Code Reference |
|----------|----------|----------------|
| Missing linguistic vars (no text in section) | Keep NaN (distinguished from 0 = no matches) | `build_linguistic_variables.py:588-589` |
| Missing CCCL_lag (calls before 2005) | Keep NaN; drops from regression sample | Expected behavior — CCCL starts 2005 |
| Missing CEO linguistic vars | Keep NaN (CEO not speaking in call) | 68.0% coverage |
| Missing controls | Drop observation in regression | `run_h6_cccl.py:161` |
| Inf values | Replace with NaN before dropna | `run_h6_cccl.py:161` |

### Lag Construction Rules

| Rule | Implementation | Code Reference |
|------|----------------|----------------|
| Consecutive fiscal years only | `NaN` if gap > 1 year between firm-year observations | `build_h6_cccl_panel.py:105-106, 113-114, 121-122` |
| Last call per fiscal year | Take CCCL value from last call (by start_date) per (gvkey, fyearq) | `build_h6_cccl_panel.py:91-95` |
| Firm-level lag | Shift within gvkey group | `build_h6_cccl_panel.py:101-104` |

### Denominator Protection

- **Linguistic percentages:** `total_tokens` in denominator; NaN if 0
- **Financial ratios:** No explicit divide-by-zero check in engine; relies on Compustat data quality
