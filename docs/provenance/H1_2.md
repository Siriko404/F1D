## A. Suite Overview

### A.1 Identity

| Field | Value |
|-------|-------|
| Hypothesis suite name/ID | H1.2 -- Financing-Constraint-Moderated Speech Uncertainty and Cash Holdings |
| Research question | Is the Manager_Pres_Uncertainty --> CashHoldings relationship stronger for financially constrained firms? Three-category test: below-IG rated vs unrated vs IG (reference). |
| Channel tested | CH1 -- Precautionary liquidity under external-finance frictions |
| Literature | Almeida, Campello & Weisbach (2004); Acharya, Almeida & Campello (2007); Bates, Kahle & Stulz (2009) |

| Primary keys | `gvkey` (entity) + `fyearq_int` (time index, Compustat fiscal year) |
| Prediction | Positive main effect of uncertainty on cash for IG firms (b1 > 0, one-tailed); interactions b4, b5 tested two-tailed (b4 != 0, b5 != 0) |
| Parent suite | H1 (Cash Holdings) -- Manager_Pres_Uncertainty_pct significant under Industry+FY FE (cols 1,3,5,7 of H1, p < 0.015 one-tailed) |

### A.2 Model Family and Estimator

| Field | Value |
|-------|-------|
| Model family | Panel FE (OLS with absorbed effects); single FE configuration |
| Estimator | `linearmodels.panel.PanelOLS`, `drop_absorbed=True`, `check_rank=False` |
| FE config | Industry FE (FF12, absorbed via `other_effects`) + Fiscal Year FE (`time_effects=True`); no entity FE |
| Within transformation | Automatic via PanelOLS (demeaning within absorbed groups) |
| Solver | Default `linearmodels` OLS solver |
| Weighting | None |
| Regularization | None |

**Model formula (5 key coefficients):**

```
CashHoldings = b1*Manager_Pres_Unc_c
             + b2*BelowIG + b3*Unrated
             + b4*(Manager_Pres_Unc_c x BelowIG)
             + b5*(Manager_Pres_Unc_c x Unrated)
             + controls + IndustryFE + FiscalYearFE + e
```

- b1 = effect for IG firms alone (reference group)
- b2 = level shift for below-IG firms at mean uncertainty
- b3 = level shift for unrated firms at mean uncertainty
- b4 = clean CH1 test: does the uncertainty-cash link strengthen for junk-rated firms?
- b5 = does it strengthen for unrated firms? (noisier, heterogeneous population)

**Rationale for Industry+FY FE only (no Firm FE):**
The parent effect is between-firm (uncertainty language varies across managers and firms). Firm FE would absorb both the IV (slow-moving within-firm) and the moderator dummies (most firms never change rating category), leaving insufficient variation for identification. This is documented in the runner docstring (lines 44-46) and is consistent with the H1 result pattern where Industry+FY FE specs show significance but Firm FE specs do not.

```python
is_ig = panel["splticrm"].isin(IG_RATINGS)
has_rating = panel["splticrm"].notna()
panel[MOD_BELOW_IG] = (has_rating & ~is_ig).astype(float)  # rated but below IG
```

**Result interpretation:**
- Neither the main IV (b1) nor either interaction (b4, b5) is statistically significant in either model.
- This is a clean null for the CH1 channel: financial constraint does not detectably amplify the uncertainty-cash link, even when below-IG and unrated firms are separated.
- **BelowIG interaction (b4) is negative in both columns** (-0.0115 in Col 1, -0.0062 in Col 2), opposite to the CH1 prediction (expected positive: constrained firms hoard more cash under uncertainty). This is a stronger null than the previous binary moderator design -- even for genuinely junk-rated firms, no amplification is detected.
- **Unrated interaction (b5) flips sign** across columns (+0.0139 in Col 1, -0.0032 in Col 2). The sign instability suggests the contemporaneous association, if any, reflects persistent cross-sectional differences rather than a dynamic effect (the positive association disappears after controlling for lagged CashHoldings in Col 2).
- **Level effects are informative:** BelowIG = +0.0304 (p < 0.001) and Unrated = +0.0632 (p < 0.001) in Col 1. Both constrained groups hold more cash than IG at mean uncertainty, with the effect twice as large for unrated firms. This validates the three-category redesign: the two groups behave differently.

**Negative within-R2 in Col 1:** Expected when Industry+FY FE absorbs more variance than the regressors explain. The overall R2 is 0.4243, indicating the model fits cross-sectional variation well. The within-R2 is the incremental explanatory power of the regressors over and above the FE-only model, which is negative because the regressors add noise after FE absorption.

## J. Known Issues / Ambiguities

### J.1 Moderator Heterogeneity (RESOLVED)

**v1 issue:** The binary `Unrated_or_NonIG` moderator conflated unrated firms (~85% of the constrained group) with below-IG rated firms (~15%). These groups have different cash-holding behaviors (unrated firms hold far more cash than below-IG firms).

**v2 status: RESOLVED.** The three-category design (BelowIG + Unrated dummies, IG reference) separates these groups. The level effects confirm the heterogeneity: BelowIG = +0.030 (p < 0.001), Unrated = +0.063 (p < 0.001) in Col 1 -- the effect is twice as large for unrated firms. This validates the redesign.

### J.2 Sample Restriction vs Parent H1

H1.2 restricts the sample to 2002-2016 (ratings coverage ends 2017-02-28), while the parent H1 uses the full 2002-2018 range. Additionally, H1.2 uses only 1 IV (Manager_Pres_Uncertainty_pct) instead of H1's 4 simultaneous IVs, which means CEO variable missingness (~30%) does not bind. This produces a much larger N (69,899 vs 57,216 in H1 Col 3). The sample difference makes direct coefficient comparison between H1 and H1.2 imprecise.

**Severity:** LOW. The year restriction is forced by data availability. The larger N from using 1 IV is a feature (more power), but it means the H1.2 "main effect" (b1) is not directly comparable to the H1 pooled coefficient.

### J.3 Negative Within-R2

Col 1 reports within-R2 = -0.1871. This is arithmetically valid under PanelOLS with `other_effects`: the within-R2 measures the incremental explanatory power of the regressors over the FE-only model. When Industry+FY FE absorbs enough between-group variance, adding regressors that are partly collinear with the FE groups can produce a negative increment. The overall R2 (0.4243) and between-R2 (0.5755) are reasonable.

**Severity:** LOW. Expected behavior for Industry FE models; the thesis should note this.

### J.4 Summary Statistics on Full Main Sample

Summary statistics (`summary_stats.csv`/`.tex`) are computed on the full Main sample (79,487 calls) rather than the regression sample (69,899 after complete-case). This means the summary stats include observations that are not in the regression due to missing controls. The difference is ~12%.

**Severity:** LOW. Common practice, but a footnote should clarify that summary stats are pre-complete-case.

### J.5 Attrition Table Structure

The attrition table mixes sequential filters (full panel -> year filter -> Main sample -> complete-case) with descriptive splits (IG / Below-IG / Unrated counts). The N Lost column for the descriptive-split rows is misleading (e.g., "Unrated firms" shows +22,422, a gain rather than a loss, because it computes the difference from the preceding row which happens to be "Below-IG firms"). This is cosmetic but could confuse readers.

**Severity:** COSMETIC.

### J.6 BelowIG Interaction Sign

The BelowIG interaction (b4) is negative in both columns (-0.0115 in Col 1, -0.0062 in Col 2), opposite to the CH1 prediction (expected positive: constrained firms hoard more cash under uncertainty). Neither is significant, but the wrong-signed point estimate for the cleanest constraint proxy (genuinely junk-rated firms) is a stronger null than the previous binary moderator design.

**Severity:** MODERATE. This is a substantive finding for the CH1 channel narrative. The thesis should discuss why even genuinely credit-constrained firms do not show amplified uncertainty-cash sensitivity.

### J.7 Unrated Interaction Sign Instability

The Unrated interaction (b5) flips sign across columns: +0.0139 (p=0.226) in Col 1 vs -0.0032 (p=0.473) in Col 2. The positive contemporaneous association disappears after controlling for lagged CashHoldings, suggesting it reflects persistent cross-sectional differences rather than a dynamic uncertainty-cash-constraint effect.

**Severity:** LOW. Neither estimate is significant; the sign instability simply confirms the null.

## K. Corporate Finance Thesis Referee Assessment

### K.1 Theoretical Motivation

**Assessment: ADEQUATE**

The CH1 channel (precautionary savings under external-finance frictions) is well-grounded in Almeida, Campello & Weisbach (2004), Acharya, Almeida & Campello (2007), and Bates, Kahle & Stulz (2009). The prediction is clear: if uncertainty language reflects genuine managerial risk perception, constrained firms should respond more strongly by hoarding cash (amplified precautionary motive). The three-category design is an improvement over the binary moderator because it separates genuinely credit-constrained firms (below-IG) from the heterogeneous unrated population.

The null result is informative and strengthened by the three-category design: even for genuinely junk-rated firms (BelowIG), the interaction is wrong-signed. This suggests the uncertainty-cash link operates through channels other than external-finance frictions.

### K.2 Identification Strategy

**Assessment: WEAK (unchanged from v1, but moderator measurement improved)**

1. **No causal identification:** Like the parent H1, this is an associational test. The interaction tests whether the association varies by financial constraint, not whether uncertainty causes cash holdings. There is no instrument, quasi-experiment, or shock-based design.

2. **VIF materially improved:** The three-category design reduced max VIF from 4.76 (binary interaction) to 3.68 (Unrated interaction) and 2.09 (BelowIG interaction). Both are well below the conventional 5.0 threshold.

3. **Cross-sectional identification only:** Most firms have zero within-firm moderator variation (rating transitions are rare). The interactions are identified from comparing constrained vs unconstrained firms within the same industry-year cell. This makes the estimates vulnerable to unobserved firm characteristics that correlate with both rating status and cash holdings.

4. **No Firm FE:** The design explicitly omits Firm FE to preserve the moderator variation. This is defensible but means the estimate absorbs all time-invariant firm-level confounds into the error term.

### K.3 Econometric Specification

**Assessment: ADEQUATE with caveats**

1. **FE choice is consistent with parent H1:** Industry+FY FE is the configuration where the parent effect is significant. Using the same configuration for the moderation test is correct.

2. **Firm-clustered SEs:** Appropriate for within-firm serial correlation in the error term.

3. **Centering:** IV is correctly mean-centered before interaction construction, ensuring interpretability of b2 and b3 at mean uncertainty.

4. **One-tailed test for main IV, two-tailed for interactions:** Reasonable. The one-tailed test for b1 follows the parent H1 directional prediction. The two-tailed test for b4 and b5 is conservative (could have been one-tailed if the prediction is specifically b4 > 0, b5 > 0).

5. **Lagged DV control (Col 2):** Including CashHoldings_t as a control in the lead specification reduces the BelowIG level effect from +0.030 to +0.003 (insignificant) and the Unrated level effect from +0.063 to +0.010, suggesting the contemporaneous specification captures persistent cash differences rather than dynamic effects.

**Caveat:** The Moulton problem noted in H1 provenance applies here too -- multiple calls per firm-quarter share the same DV, inflating effective N. Firm-clustered SEs mitigate but do not fully resolve this.

### K.4 Data Quality and Measurement

**Assessment: GOOD**

1. **Ratings data:** Compustat Daily Ratings is a standard source. The merge_asof approach correctly avoids look-ahead bias. The 22 rating codes match expected S&P rating scales.

2. **IG threshold at BBB-:** This is the standard investment-grade boundary in the corporate finance literature (e.g., Kisgen 2006, 2009). No issue.

3. **Three-category classification is correct:** Verified that (a) categories are mutually exclusive and exhaustive, (b) BelowIG captures firms rated BB+ through SD, (c) Unrated captures firms with no rating match, and (d) IG captures BBB- and above.

4. **BelowIG group size is adequate:** N_below_ig = 16,214 in Col 1 (23.2% of regression sample). This is sufficient for identification.

5. **Panel inheritance:** The H1 panel is well-documented and audited (see H1.md). No additional data quality concerns specific to H1.2.

### K.5 Result Interpretation

**Assessment: GOOD**

1. **Clean null result with stronger evidence:** Neither the main IV (b1) nor either interaction (b4, b5) is significant in either model. The three-category design provides a more convincing null than the binary moderator because (a) the cleanest constraint proxy (BelowIG) is wrong-signed, and (b) the noisier proxy (Unrated) is sign-unstable across columns.

2. **Level effects validate the three-category design:** BelowIG = +0.030 (p < 0.001) and Unrated = +0.063 (p < 0.001) in Col 1 confirm that both constrained groups hold more cash than IG at mean uncertainty, with the Unrated effect twice as large. The v1 audit's concern about heterogeneity (Issue J.1) was well-founded and is now resolved.

3. **Lagged DV model:** In Col 2, the lagged DV (b = 0.766) absorbs most of the cross-sectional variation, pushing the BelowIG level effect to near-zero (0.003, p=0.158). This suggests the Col 1 level effects reflect persistent cash differences rather than a dynamic uncertainty effect.

4. **b4 interpretation for thesis:** b4 = -0.012 (p=0.24) in Col 1. The CH1 prediction is b4 > 0. Even the point estimate is negative. For the cleanest constraint proxy (genuinely junk-rated firms), there is no evidence of amplified uncertainty-cash sensitivity. This is a clear non-finding for the specific channel.

### K.6 Robustness and Alternative Explanations

**Assessment: IMPROVED (some v1 items addressed, others remain)**

1. **Separate unrated from below-IG (v1 item):** RESOLVED by three-category design.

2. **VIF improvement:** Max VIF reduced from 4.76 to 3.68. This is a material improvement worth noting in the thesis.

Remaining robustness checks that a thesis committee would likely request:

3. **Alternative constraint measures:** Whited-Wu index, SA index, or Hadlock-Pierce index would provide continuous constraint proxies. If the SA-based interaction is also null, the null result is more convincing.

4. **Size interaction:** A referee may still want to see a Size x Uncertainty interaction to verify the constraint effect is not confounded with firm-size effects.

5. **Rating transition analysis:** Restrict to firms with within-firm moderator variation and examine whether transitions to/from IG status change the uncertainty-cash sensitivity.

### K.7 Overall Readiness

**Assessment: SUITABLE FOR THESIS WITH CAVEATS**

The H1.2 suite is technically well-executed, reproducible, and properly documented. The three-category design is a genuine improvement over the binary moderator (resolves moderator heterogeneity, reduces VIF). The null result is informative, honest, and now more convincing: even the cleanest constraint proxy (BelowIG) shows no amplification. A thesis committee will likely accept the null, but may request one alternative constraint measure (e.g., SA index) to rule out measurement artifacts.
