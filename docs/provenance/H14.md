# H14: Speech Uncertainty and Bid-Ask Spread Changes — Data Provenance

**Suite:** H14 Bid-Ask Spread
**Status:** Redesigned (2026-03-17)
**Estimation Suite Type:** Panel Fixed Effects OLS

## A. Suite Identity

| Field | Value |
|-------|-------|
| **Hypothesis** | H14: β(uncertainty) > 0 — higher speech uncertainty predicts wider bid-ask spreads |
| **DV** | DSPREAD — change in average relative bid-ask spread around the call |
| **DV Construction** | Lee (2016, *The Accounting Review*): RelSpread_d = 2×(ASK−BID)/(ASK+BID) using closing quotes; DSPREAD = mean(RelSpread[+1,+3]) − mean(RelSpread[−3,−1]) in trading days |
| **Unit** | Individual earnings call (file_name) |
| **Panel Type** | Unbalanced panel with Industry/Firm + FiscalYear fixed effects |
| **Sample** | Main only (FF12 ∉ {8, 11}) |
| **N (regression)** | 37,657 calls, 1,276 firms |
| **Model Family** | Panel Fixed Effects OLS (linearmodels PanelOLS) |
| **Time FE** | `fyearq_int` (fiscal year) |
| **SEs** | Firm-clustered |
| **P-values** | One-tailed (H14: β > 0) |

## B. Key IVs (6 simultaneous)

All 6 IVs enter every model simultaneously (horse-race):

| IV | Source | Coverage |
|----|--------|----------|
| CEO_QA_Uncertainty_pct | LinguisticEngine (Stage 2) | 70.4% |
| CEO_Pres_Uncertainty_pct | LinguisticEngine (Stage 2) | 70.3% |
| Manager_QA_Uncertainty_pct | LinguisticEngine (Stage 2) | 95.8% |
| Manager_Pres_Uncertainty_pct | LinguisticEngine (Stage 2) | 97.5% |
| CEO_Clarity_Residual | H0.3 regression residual | 43.8% |
| Manager_Clarity_Residual | H0.3 regression residual | 60.2% |

**Note on clarity residuals:** Expected to have *negative* coefficients (more clarity → narrower spreads). The one-tailed β > 0 test is conservative for these by design, matching all other suites (H1/H4/H5).

### FE Implementation
- **Industry FE (cols 1, 3):** `PanelOLS(other_effects=ff12_code, entity_effects=False, time_effects=True, check_rank=False)`
- **Firm FE (cols 2, 4):** `PanelOLS.from_formula("DSPREAD ~ 1 + {IVs + controls} + EntityEffects + TimeEffects")`
- **Time index:** `set_index(["gvkey", "fyearq_int"])`

## E. Results

| IV | Col 1 | Col 2 | Col 3 | Col 4 | Summary |
|----|-------|-------|-------|-------|---------|
| CEO QA Unc | −0.0001 | −0.0003 | −0.0002 | −0.0003 | 0/4 sig (wrong sign) |
| CEO Pres Unc | −0.0000 | 0.0000 | −0.0000 | 0.0000 | 0/4 sig |
| Mgr QA Unc | 0.0002** | 0.0001 | 0.0005*** | 0.0001 | 2/4 sig (industry FE only) |
| Mgr Pres Unc | −0.0000 | 0.0000 | 0.0000 | 0.0000 | 0/4 sig |
| CEO Clarity | 0.0001** | 0.0003*** | 0.0002*** | 0.0003*** | 4/4 sig |
| Mgr Clarity | −0.0002 | −0.0002 | −0.0005 | −0.0001 | 0/4 sig (expected negative) |

**Key findings:**
1. **CEO Clarity Residual:** Strongly significant across all 4 specs (β > 0, p < 0.01 in cols 2-4). *Surprising:* positive coefficient means higher clarity residual → wider spreads, opposite to theoretical expectation.
2. **Manager QA Uncertainty:** Significant in industry FE specs (cols 1, 3) but not firm FE (cols 2, 4). Suggests between-firm variation drives the effect.
3. **All other IVs:** Null results.

Within-R² ranges 0.327–0.349, driven primarily by PreCallSpread (lagged DV control).
