## A. Suite Overview

### A1) Identity

| Field | Value |
|-------|-------|
| **Suite Name** | H14 Bid-Ask Spread Change |
| **Research Question** | Is higher earnings-call language uncertainty associated with a larger increase in bid-ask spreads around the conference call (i.e., lower market liquidity)? |
| **Estimation Unit** | Call-level (individual earnings call) |
| **Primary Keys** | `file_name` (call identifier), `gvkey` (firm), `year_quarter` (time) |

### A2) Model Family and Estimator

| Field | Value |
|-------|-------|
| **Model Family** | Panel OLS with Fixed Effects |
| **Estimator** | `linearmodels.PanelOLS` with `EntityEffects + TimeEffects` |
| **Objective** | Least squares with within-transformation |
| **Weighting** | None (unweighted) |
| **Software** | Python `linearmodels` v0.6.0+ |

### A3) Outcome / Target Definition

| Variable | Definition | Timing | Missingness |
|----------|------------|--------|-------------|
| `delta_spread` | Change in relative bid-ask spread: `PostSpread - PreSpread` where `Spread_d = 2 * (ASKHI - BIDLO) / (ASKHI + BIDLO)` | Pre = trading days -3,-2,-1; Post = +1,+2,+3 relative to call | Dropped if NaN; requires ≥2 valid days in each window; ~98.9% coverage |

**Construction Algorithm:**
1. For each call, identify PERMNO via date-bounded CCM linkage (`linkdt ≤ call_date ≤ linkenddt`)
2. Find reference date = last trading day on or before call date
3. PreSpread = mean(spread) for 3 trading days before reference date (positions -3, -2, -1)
4. PostSpread = mean(spread) for 3 trading days after reference date (positions +1, +2, +3)
5. delta_spread = PostSpread - PreSpread

### A4) Main RHS and Controls

**Independent Variables (6 measures, each tested individually):**

| Variable | Description |
|----------|-------------|
| `Manager_QA_Uncertainty_pct` | Manager Q&A language uncertainty (primary) |
| `CEO_QA_Uncertainty_pct` | CEO Q&A language uncertainty |
| `Manager_Pres_Uncertainty_pct` | Manager Presentation language uncertainty |
| `CEO_Pres_Uncertainty_pct` | CEO Presentation language uncertainty |
| `Manager_Clarity_Residual` | Manager Q&A uncertainty residual (after firm/linguistic controls) |
| `CEO_Clarity_Residual` | CEO Q&A uncertainty residual (after firm/linguistic controls) |

**Control Variables:**

| Variable | Description | Source |
|----------|-------------|--------|
| `Size` | Log total assets (`ln(atq)`) | Compustat |
| `StockPrice` | Stock price at call date (nearest prior trading day) | CRSP |
| `Turnover` | Share turnover (`VOL / (SHROUT * 1000)`) | CRSP |
| `Volatility` | Annualized return volatility over inter-call window | CRSP |
| `PreCallSpread` | Pre-call average bid-ask spread (control for baseline liquidity) | CRSP |
| `AbsSurprise` | Absolute earnings surprise (`|ACTUAL - MEANEST| / |MEANEST|`) | IBES |

**Sample Filters:**
- `min_calls >= 5` — Firms with fewer than 5 calls excluded from regression
- Industry sample: Main (FF12 ≠ 8,11) — only Main sample used in regressions
