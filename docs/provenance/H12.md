## A. Suite Overview

### A1) Identity

| Field | Value |
|-------|-------|
| **Hypothesis Suite Name/ID** | H12 Payout Ratio |
| **Research Question** | Does speech uncertainty predict payout ratio? Tests whether managerial uncertainty in earnings calls predicts contemporaneous and forward dividend payout ratios. |
| **Estimation Unit** | Firm-fiscal-year (gvkey, fyearq_int) |
| **Unique Record Identifier** | (gvkey, fyearq_int) -- unique firm-year |
| **Panel Index** | `gvkey` (entity) + `fyearq_int` (fiscal year time) |
| **Panel Type** | Unbalanced panel with Industry/Firm + FiscalYear fixed effects |

| Field | Value |

| **Weighting** | None (unweighted) |

### A3) Outcome / Target Definition

| Variable | Description | Timing | Notes |
|----------|-------------|--------|-------|
| `PayoutRatio` | DVC / IB = dvy / iby (Attig et al.) | Contemporaneous (t) | NaN when iby <= 0 (negative earnings) |
| `PayoutRatio_lead` | PayoutRatio shifted forward one fiscal year | Forward (t+1) | Requires consecutive fiscal years |

**DV Construction (Attig et al.):**
- DVC = `dvy` (annual common dividends, Q4 cumulative from Compustat)
- IB = `iby` (income before extraordinary items, annual, Q4 cumulative)
- PayoutRatio = dvy / iby when iby > 0; NaN when iby <= 0
- Missing dvy with iby > 0 treated as PayoutRatio = 0 (firm has income but pays no dividends)

Aggregation rule: arithmetic mean across all quarterly earnings calls within the fiscal year (per Brochet et al.; Hassan et al. QJE).

## B. Redesign Rationale

The old H12 used DivIntensity = dvy/atq (dividends/assets) as the DV -- this is not the standard dividend measure in the literature. The redesign adopts **PayoutRatio = DVC/IB** per Attig et al., which is the canonical payout ratio measure.

Additional changes:
- 6 simultaneous IVs (replaces single-IV-per-regression architecture)
- Clarity Residuals replace Weak Modal measures
- Industry/Firm FE crossed with Base/Extended controls (replaces Firm+Year FE only)
- Main sample only (replaces 3-sample design)
- Fiscal year time index (fixes calendar year FE bug)
- No MIN_CALLS_PER_FIRM filter (was misleadingly named, removed)
