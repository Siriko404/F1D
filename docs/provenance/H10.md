## A. Suite Overview

### A1) Identity

| Field | Value |
|-------|-------|
| **Hypothesis Suite ID** | H10 (H_TT: Tone-at-the-Top) |
| **Research Question** | Does CEO communicative uncertainty transmit to subordinate managers? Specifically: (1) Does CEO style (real-time clarity) predict CFO uncertainty? (2) Do prior CEO Q&A turns predict non-CEO manager turn-level uncertainty? |
| **Estimation Unit** | M1: Call-level (firm-quarter); M2: Speaker-turn level within Q&A |
| **Primary Keys** | M1: `file_name` (unique call ID), linked to `gvkey` + `ceo_id`; M2: (`file_name`, `turn_index`, `speaker_id`) |

### A2) Model Family and Estimator

| Model | Family | Estimator | Details |
|-------|--------|-----------|---------|
| **M1 (Call-level)** | Panel OLS with Fixed Effects | `linearmodels.panel.PanelOLS` | Entity (Firm) + Time (Year-Quarter) FE; two-way clustered SEs (Firm × CEO) |
| **M2 (Turn-level)** | Absorbing Least Squares | `linearmodels.iv.absorbing.AbsorbingLS` | Call FE + Speaker FE absorbed; two-way clustered SEs (Firm × Call) |

**Weighting:** None (unweighted OLS)

### A3) Outcome(s) / Target Definition

| Model | Dependent Variable | Coding Rules | Timing |
|-------|-------------------|--------------|--------|
| M1 | `IHS_CFO_QA_Unc` | IHS transform of `CFO_QA_Uncertainty_pct` (CFO-specific uncertainty % in Q&A); IHS: `arcsinh(x)` | Measured at call `start_date` (time t) |
| M2 | `IHS_NonCEO_Turn_Unc` | IHS transform of `Turn_Uncertainty_pct` for each non-CEO manager turn; IHS: `arcsinh(x)` | Measured at turn `turn_index` within call |

**IHS Transformation:** `asinh(x) = ln(x + sqrt(x^2 + 1))` — handles zero values and right-skewed distributions without dropping zeros like log.

### A4) Main RHS and Controls

**M1 (Call-level) Primary Controls (no CEO same-call):**
- `ClarityStyle_Realtime` — Main IV: Real-time CEO style (4-call rolling window, EB-shrunk)
- `Size` — Log total assets
- `BM` — Book-to-market ratio
- `Lev` — Leverage ratio
- `ROA` — Return on assets
- `StockRet` — Stock return (inter-call window)
- `MarketRet` — Market return (inter-call window)
- `EPS_Growth` — EPS growth (YoY)
- `SurpDec` — Earnings surprise decile (-5 to +5)

**M2 (Turn-level) Specifications:**
- Primary IV: `IHS_CEO_Prior_QA_Unc` — IHS of expanding mean CEO uncertainty from prior Q&A turns in same call
- Robustness IVs: `IHS_CEO_Unc_Lag1`, `IHS_CEO_Unc_Roll2`, `IHS_CEO_Unc_Roll3`, `IHS_CEO_Unc_ExpDecay`, `IHS_CEO_Unc_Lead1` (placebo)
- Optional controls: `turn_index`, `turn_index_sq`, `IHS_Analyst_Unc`

### A5) Fixed Effects + Variance Estimator

| Model | Fixed Effects | Variance Estimator |
|-------|--------------|-------------------|
| M1 | Firm (`gvkey`) + Year-Quarter (`yq_id`) | Two-way clustered: Firm × CEO |
| M2 | Call (`file_name`) + Speaker (`speaker_id`) | Two-way clustered: Firm × Call |

**Note on M2 clustering:** Per Addendum C, M2 clusters by (Firm, Call) rather than (Firm, CEO) because observations are nested within calls.

### A6) Model-Family Addendum

#### OLS/Panel (M1, M2)
- **Link/Transformation:** IHS (inverse hyperbolic sine) for DV and key IVs
- **Within transformation:** Entity + time dummies absorbed via `drop_absorbed=True`
- **Small-sample corrections:** Wild cluster bootstrap for Finance/Utility samples (n_bootstrap=9999, seed=42, Rademacher weights)

## G. Outliers, Missing Data, Scaling, and Model-Specific Data Rules

### Winsorization

| Variable | Winsorization Method | Thresholds | Where Applied | Notes |
|----------|---------------------|------------|---------------|-------|
| CEO_QA_Uncertainty_pct | Per-year 1%/99% | Lower=0.01, Upper=0.99 | LinguisticEngine | Max varies by year (1.89-2.51) |
| CFO_QA_Uncertainty_pct | Pooled 1%/99% | Lower=0.01, Upper=0.99 | `CFOQAUncertaintyBuilder._finalize_data()` | Pooled max=3.378 |
| Turn_Uncertainty_pct | None (intentional) | N/A | N/A | IHS transform handles extremes; extreme values (>50%) retained as valid signals of short-turn uncertainty; Call + Speaker FE absorb turn-level outliers |
| Financial controls (Size, BM, Lev, ROA) | Per-year 1%/99% | Lower=0.01, Upper=0.99 | CompustatEngine | Standard Compustat processing |
| Returns (StockRet, MarketRet) | No winsorization | — | Raw values used | Inter-call compound returns |

### Missing Data Policy

| Variable | Missingness (Call Panel) | Policy |
|----------|-------------------------|--------|
| `ClarityStyle_Realtime` | 44.5% | Listwise deletion in regression (NaN → obs dropped) |
| `CFO_QA_Uncertainty_pct` | 12.9% | Listwise deletion (no CFO in QA for some calls) |
| `SurpDec` | 25.5% | Listwise deletion (no IBES match) |
| `Size`, `Lev` | 0.2% | Listwise deletion (Compustat link failure) |
| `StockRet`, `MarketRet` | 6.7% | Listwise deletion (insufficient trading days) |

### IHS Transformation Rationale

- Handles zero values (unlike log which requires x > 0)
- Approximates log for large values: `asinh(x) ≈ ln(2x)` for x >> 1
- Applied to percentage variables that can be zero

### 1. Significant Placebo Test in M2

**Issue:** The M2 placebo specification (`M2_placebo_lead`) using future CEO uncertainty (IHS_CEO_Unc_Lead1) shows a highly significant positive coefficient (0.0373, t=31.49, p<0.001).

**Impact:** This suggests that future CEO uncertainty also predicts current manager uncertainty, which may indicate:
1. Within-call uncertainty clustering (calls have overall high/low uncertainty)
2. Time trends within calls affecting both CEO and managers
3. Potential confounding by call-level factors not fully absorbed by Call FE

**Recommended Fix:** The significant placebo may reflect legitimate within-call dynamics rather than a methodological flaw, but interpretation of the main M2 results should acknowledge this pattern. Consider explicit modeling of call-level uncertainty shocks.

**Status:** KNOWN — placebo test implemented and documented; results suggest within-call correlation structure warrants attention.
