## A. Suite Overview

### A1) Identity

### A2) Model Family and Estimator

| Field | Value |

### A3) Outcome / Target Definition

| Variable | Definition | Timing | Missingness |
|----------|------------|--------|-------------|
| `delta_dispersion` | Change in analyst dispersion = `dispersion_{t+3} - dispersion_{t-3}` (3 trading days after minus before call) | Window around call date | Dropped if NaN; coverage varies |
| `dispersion_before` | Analyst dispersion computed from estimates active 3 trading days before call | `call_date - 3 trading days` | Can be missing if insufficient analysts |
| `dispersion_after` | Analyst dispersion computed from estimates active 3 trading days after call | `call_date + 3 trading days` | Can be missing if insufficient analysts |
| `dispersion` | Legacy: current period (t) analyst dispersion (backward merge) | IBES consensus on/before call | ~77% coverage (for comparison) |

**Dispersion Computation Method:**
```
dispersion = STDEV(VALUE) / |MEAN(VALUE)|
```
Where VALUEs are individual analyst estimates for the target fiscal quarter (FPEDATS), filtered to:
1. Estimates active on/before the target date
2. Stale estimate filter: only estimates updated within 180 days
3. Most recent estimate per analyst
4. Minimum 2 analysts required

### A4) Model Specifications

**Model A (Uncertainty → Dispersion) — 4 Regressions per Sample:**

| Spec ID | Key IV | Description |

| A3 | `Manager_QA_Uncertainty_pct` | Manager Q&A uncertainty |
| A4 | `Manager_Pres_Uncertainty_pct` | Manager Presentation uncertainty |

**Control Variables:**

*Linguistic Controls:*
- `Analyst_QA_Uncertainty_pct` — Analyst Q&A uncertainty
- `Entire_All_Negative_pct` — Entire call negative sentiment ("negative all")

*Financial Controls:*
- `Size` — Log total assets (Compustat)
- `Lev` — Total debt / Total assets (Compustat)
- `TobinsQ` — `(atq + cshoq*prccq - ceqq) / atq` (Compustat)
- `earnings_volatility` — Rolling 5yr std of earnings/assets (Compustat)
- `earnings_surprise_ratio` — `|ACTUAL - MEANEST| / |MEANEST|` from IBES
- `loss_dummy` — `1` if `ibq < 0`, else `0` (Compustat)

*Lagged DV Control:*
- `lagged_dispersion` — Prior period (t-1) analyst dispersion

**Sample Filters:**
- `min_calls >= 5` — Firms with fewer than 5 calls excluded from regression

### IBES Merge Logic

| Variable | Merge Method | Tolerance | Direction |
|----------|--------------|-----------|-----------|
| `delta_dispersion` | Computed from IBES Detail | — | ±3 trading days around call |
| `dispersion_before` | Point-in-time computation | — | `call_date - 3 trading days` |
| `dispersion_after` | Point-in-time computation | — | `call_date + 3 trading days` |
| `dispersion` | `pd.merge_asof` | 365 days | **backward** (legacy, for comparison) |
| `lagged_dispersion` | Computed from prior consensus | — | t-1 (one consensus before t) |
| `earnings_surprise_ratio` | `pd.merge_asof` | None | **backward** |

**Dispersion Computation Algorithm:**
1. For each call, identify target fiscal quarter (FPEDATS closest to call date)
2. Filter to estimates for that gvkey/quarter
3. At each target date (before/after):
   - Filter to estimates active on/before target date
   - Apply stale estimate filter (max 180 days old)
   - Keep most recent estimate per analyst
   - Require minimum 2 analysts
   - Compute dispersion = STDEV(VALUE) / |MEAN(VALUE)|
4. delta_dispersion = dispersion_after - dispersion_before

**Key Linking:** IBES CUSIP8 → CCM CUSIP8 → GVKEY (6-digit, zero-padded)
