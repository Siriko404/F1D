## A. Suite Overview

### A1) Identity

| Field | Value |
|-------|-------|
| **Suite Name** | H5 Analyst Dispersion |
| **Research Question** | Does managerial speech vagueness (hedging language / QA-Pres uncertainty gap) predict higher analyst forecast dispersion at t+1? |
| **Estimation Unit** | Call-level (individual earnings call) |
| **Primary Keys** | `file_name` (call identifier), `gvkey` (firm), `year` |

### A2) Model Family and Estimator

| Field | Value |
|-------|-------|
| **Model Family** | Panel OLS with Fixed Effects |
| **Estimator** | `linearmodels.PanelOLS` with `entity_effects=True`, `time_effects=True` |
| **Objective** | Least squares with within-transformation |
| **Weighting** | None (unweighted) |
| **Software** | Python `linearmodels` v0.6.0+ |

### A3) Outcome / Target Definition

| Variable | Definition | Timing | Missingness |
|----------|------------|--------|-------------|
| `dispersion_lead` | t+1 analyst forecast dispersion = `STDEV / |MEANEST|` from IBES | First IBES consensus strictly AFTER call date (forward merge_asof, tolerance=180 days) | Dropped if NaN; 75.3% coverage |

### A4) Main RHS and Controls

**Key Independent Variables:**

| Model | Target Variable | Base Variable |
|-------|-----------------|---------------|
| Model A (Hedging) | `Manager_QA_Weak_Modal_pct` | `Manager_QA_Uncertainty_pct` |
| Model B (Gap) | `Uncertainty_Gap` (= `Manager_QA_Uncertainty_pct` - `Manager_Pres_Uncertainty_pct`) | `Manager_Pres_Uncertainty_pct` |

**Control Block (BASE_CONTROLS):**
- `Analyst_QA_Uncertainty_pct` — Analyst Q&A uncertainty (linguistic)
- `earnings_surprise_ratio` — `|ACTUAL - MEANEST| / |MEANEST|` from IBES
- `loss_dummy` — `1` if `ibq < 0`, else `0` (Compustat)
- `Size` — Log total assets (Compustat)
- `Lev` — Total debt / Total assets (Compustat)
- `TobinsQ` — `(atq + cshoq*prccq - ceqq) / atq` (Compustat)
- `earnings_volatility` — Rolling 5yr std of earnings/assets (Compustat)

**Optional Lagged DV (Model specs with "Lagged DV"):**
- `prior_dispersion` — Current quarter (t) analyst dispersion = `STDEV / |MEANEST|`

**Sample Filters:**
- `min_calls >= 5` — Firms with fewer than 5 calls excluded from regression

### A5) Fixed Effects & Variance Estimator

| Field | Value |
|-------|-------|
| **Entity FE** | `C(gvkey)` — Firm fixed effects |
| **Time FE** | `C(year)` — Year fixed effects |
| **Standard Errors** | Clustered at firm level (`cluster_entity=True`) |
| **Small-sample Correction** | `debiased=True` (linearmodels default) |

### A6) Model-Family Addendum

**N/A** — Standard panel OLS; no survival, IV, DiD, or GLM features.

## G. Outliers, Missing Data, Scaling, and Model-Specific Data Rules

### Winsorization

| Engine | Method | Variables | Thresholds | When Applied |
|--------|--------|-----------|------------|--------------|
| **CompustatEngine** | Per-year | Size, Lev, ROA, CurrentRatio, RD_Intensity, CashHoldings, TobinsQ, etc. | 1%/99% per-year | At engine load time |
| **LinguisticEngine** | Per-year | All `_pct` columns (25+) | 0%/99% per-year (lower=0.0) | At engine load time |
| **IbesEngine** | Pooled | `dispersion`, `earnings_surprise_ratio` | 1%/99% pooled | After aggregation in engine |

### Missing Data Policy

| Scenario | Handling | Code Reference |
|----------|----------|----------------|
| Missing linguistic vars (no text in section) | Keep NaN (distinguished from 0 = no matches) | `build_linguistic_variables.py:588-589` |
| Missing dispersion_lead | Drop observation in regression (listwise deletion) | `run_h5_dispersion.py:154` |
| Missing controls | Drop observation in regression | `run_h5_dispersion.py:154` |
| Inf values | Replace with NaN before dropna | `run_h5_dispersion.py:154` |

### Denominator Protection

- **Linguistic percentages:** `total_tokens` in denominator; NaN if 0
- **IBES dispersion:** `|MEANEST|` must be ≥ 0.05; filtered at engine level (`_ibes_engine.py:108`)
- **Financial ratios:** No explicit divide-by-zero check in engine; relies on Compustat data quality

### Scaling / Standardization

- **None applied** in H5. Variables used in levels (log-transformed where noted).
