## A. Suite Overview

### A1) Identity

| Field | Value |
|-------|-------|
| **Suite Name** | H5 Analyst Dispersion |
| **Research Question** | Does managerial speech vagueness (uncertainty language / Pres-QA gaps) predict contemporaneous analyst forecast dispersion? |
| **Estimation Unit** | Call-level (individual earnings call) |
| **Primary Keys** | `file_name` (call identifier), `gvkey` (firm), `year` |

### A2) Model Family and Estimator

| Field | Value |
|-------|-------|
| **Model Family** | Panel OLS with Fixed Effects |
| **Estimator** | `linearmodels.PanelOLS` with `entity_effects=True`, `time_effects=True` |
| **Objective** | Least squares with within-transformation |
| **Weighting** | None (unweighted) |
| **Software** | Python `linearmodels` v0.6.0+ |

### A3) Outcome / Target Definition

| Variable | Definition | Timing | Missingness |
|----------|------------|--------|-------------|
| `dispersion` | Current period (t) analyst forecast dispersion = `STDEV / |MEANEST|` from IBES | IBES consensus on or before call date (backward merge_asof, tolerance=365 days) | Dropped if NaN; ~77% coverage |

### A4) Model Specifications

**Model A (Uncertainty → Dispersion) — 4 Regressions per Sample:**

| Spec ID | Key IV | Description |
|---------|--------|-------------|
| A1 | `CEO_QA_Uncertainty_pct` | CEO Q&A uncertainty |
| A2 | `CEO_Pres_Uncertainty_pct` | CEO Presentation uncertainty |
| A3 | `Manager_QA_Uncertainty_pct` | Manager Q&A uncertainty |
| A4 | `Manager_Pres_Uncertainty_pct` | Manager Presentation uncertainty |

**Model B (Gaps → Dispersion) — 4 Regressions per Sample:**

| Spec ID | Key IV | Gap Formula (Pres - QA) | Description |
|---------|--------|-------------------------|-------------|
| B1 | `CEO_Pres_QA_Gap` | `CEO_Pres_Uncertainty_pct - CEO_QA_Uncertainty_pct` | CEO Pres-QA gap |
| B2 | `Mgr_Pres_QA_Gap` | `Manager_Pres_Uncertainty_pct - Manager_QA_Uncertainty_pct` | Mgr Pres-QA gap |
| B3 | `CEO_Mgr_QA_Gap` | `CEO_QA_Uncertainty_pct - Manager_QA_Uncertainty_pct` | CEO-Mgr QA gap (regime) |
| B4 | `CEO_Mgr_Pres_Gap` | `CEO_Pres_Uncertainty_pct - Manager_Pres_Uncertainty_pct` | CEO-Mgr Pres gap (regime) |

**Gap Direction:** Positive gap = more uncertain in prepared remarks (B1-B2) or CEO more uncertain than broader team (B3-B4).

**Control Variables (Both Models):**

*Linguistic Controls:*
- `Analyst_QA_Uncertainty_pct` — Analyst Q&A uncertainty
- `Entire_All_Negative_pct` — Entire call negative sentiment ("negative all")

*Financial Controls:*
- `Size` — Log total assets (Compustat)
- `Lev` — Total debt / Total assets (Compustat)
- `TobinsQ` — `(atq + cshoq*prccq - ceqq) / atq` (Compustat)
- `earnings_volatility` — Rolling 5yr std of earnings/assets (Compustat)
- `earnings_surprise_ratio` — `|ACTUAL - MEANEST| / |MEANEST|` from IBES
- `loss_dummy` — `1` if `ibq < 0`, else `0` (Compustat)

*Lagged DV Control:*
- `lagged_dispersion` — Prior period (t-1) analyst dispersion

**Sample Filters:**
- `min_calls >= 5` — Firms with fewer than 5 calls excluded from regression

### A5) Fixed Effects & Variance Estimator

| Field | Value |
|-------|-------|
| **Entity FE** | `C(gvkey)` — Firm fixed effects |
| **Time FE** | `C(year)` — Year fixed effects |
| **Standard Errors** | Clustered at firm level (`cluster_entity=True`) |
| **Small-sample Correction** | `debiased=True` (linearmodels default) |

### A6) Model-Family Addendum

**N/A** — Standard panel OLS; no survival, IV, DiD, or GLM features.
