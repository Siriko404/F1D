# H8: CEO Speech Vagueness Moderates Political Risk and Abnormal Investment

---

## A. Suite Overview

### A1) Identity

| Field | Value |
|-------|-------|
| **Suite ID** | H8 |
| **Suite Name** | Political Risk × CEO Speech Vagueness → Abnormal Investment |
| **Research Question** | Does CEO communication clarity moderate the effect of firm-level political risk exposure on abnormal (over/under) investment? |
| **Estimation Unit** | Firm-fiscal-year (gvkey, fyearq) |
| **Primary Keys** | gvkey (firm identifier), fyearq (fiscal year) |

### A2) Model Family and Estimator

| Field | Value |
|-------|-------|
| **Model Family** | Panel Fixed Effects OLS |
| **Estimator** | PanelOLS (linearmodels) with EntityEffects + TimeEffects |
| **Objective** | Least squares with within-transformation |
| **Weighting** | None (unweighted) |

### A3) Outcome / Target Definition

| Field | Value |
|-------|-------|
| **Dependent Variable** | AbsAbInv_lead = \|InvestmentResidual\|_{t+1} |
| **Coding Rules** | Absolute value of Biddle (2009) investment residual, shifted forward one fiscal year |
| **Timing Convention** | DV at t+1 (forward-looking); all RHS at t |
| **Missingness Rule** | Listwise deletion for regression; gap years (non-consecutive fyearq) set to NaN (no survivorship selection) |

### A4) Main RHS and Controls

| Category | Variables |
|----------|-----------|
| **Key Independent Variables** | PRiskFY (political risk exposure), style_frozen (CEO clarity score) |
| **Interaction Term** | interact = PRiskFY × style_frozen |
| **Controls** | Size, Lev, ROA, TobinsQ (firm-year level from Compustat) |
| **Sample Filters** | Minimum 5 calls per CEO (enforced in CEO Clarity builder) |

### A5) Fixed Effects + Variance Estimator

| Field | Value |
|-------|-------|
| **Fixed Effects** | Firm FE (EntityEffects) + Year FE (TimeEffects) |
| **Variance Estimator** | Clustered standard errors at firm level (cluster_entity=True) |
| **Small Sample Corrections** | None explicitly specified |

### A6) Model-Family Addendum

**N/A** — Standard panel OLS, no survival/IV/DiD/RD/time-series/GLM specifics apply.

## C. Dependency Chain (End-to-End)

### Step 1: Raw Data Sources

| Source | Location | Description |
|--------|----------|-------------|
| Compustat | `inputs/comp_na_daily_all/comp_na_daily_all.parquet` | Quarterly financial data (467MB) |
| Hassan PRisk | `inputs/FirmLevelRisk/firmquarter_2022q1.csv` | Hassan et al. (2019) political risk scores |
| CEO Clarity | `outputs/econometric/ceo_clarity/{timestamp}/clarity_scores.parquet` | Phase 56 CEO clarity scores |
| Earnings Calls Manifest | `outputs/1.4_AssembleManifest/{timestamp}/master_sample_manifest.parquet` | Call-level metadata with gvkey, ceo_id |

### Step 2: Panel Building (Stage 3)

**Script:** `src/f1d/variables/build_h8_political_risk_panel.py`

1. **Load manifest** — call-level data with identifiers (112,968 rows → filtered to year range)
2. **Load builders:**
   - `ManifestFieldsBuilder` — file_name, gvkey, ceo_id, start_date, ff12_code
   - `SizeBuilder` — Size = ln(atq)
   - `LevBuilder` — Lev = (dlcq + dlttq) / atq
   - `ROABuilder` — ROA = iby_annual / avg_assets
   - `TobinsQBuilder` — TobinsQ = (atq + cshoq×prccq - ceqq) / atq
   - `InvestmentResidualBuilder` — Biddle (2009) abnormal investment
   - `PRiskFYBuilder` — Hassan PRisk fiscal-year aggregate
   - `CEOClarityStyleBuilder` — frozen CEO clarity per firm-year
3. **Merge all builders** — left join on file_name (call-level)
4. **Attach fyearq** — via `panel_utils.attach_fyearq()` using Compustat datadate
5. **Aggregate to firm-year** — `aggregate_to_firm_year()` takes last non-missing per (gvkey, fyearq)
6. **Create forward DV** — `create_lead_absabinv()` shifts AbsAbInv one year forward, nulls gap years
7. **Create interaction** — `create_interaction()` computes PRiskFY × style_frozen

### Step 3: Estimation (Stage 4)

**Script:** `src/f1d/econometric/run_h8_political_risk.py`

1. Load firm-year panel from Stage 3
2. Generate summary statistics
3. Run sanity checks (distributions, coverage)
4. Estimate PanelOLS:
   - Formula: `AbsAbInv_lead ~ PRiskFY + style_frozen + interact + Size + Lev + ROA + TobinsQ + EntityEffects + TimeEffects`
   - Clustered SEs at firm level
5. Output LaTeX table and diagnostics

## G. Outliers, Missing Data, Scaling, and Model-Specific Data Rules

### Winsorization

| Variable | Threshold | Method | Where Applied |
|----------|-----------|--------|---------------|
| Lev, ROA, TobinsQ, CashFlow, etc. | 1%/99% | Pooled (all years) | `_compustat_engine.py:1050-1057` |

### Missing Data Policy

| Scenario | Policy |
|----------|--------|
| Missing Compustat values | NaN propagated; listwise deletion in regression |
| Missing PRiskFY | NaN if <2 quarters in 366-day window (no forward-fill) |
| Missing style_frozen | NaN if CEO not matched or <5 total calls |
| Gap years in DV | AbsAbInv_lead set to NaN (no interpolation) |

### Denominator Protections

| Variable | Protection |
|----------|------------|
| Size | `ln(atq)` only for atq > 0; zero/negative → NaN |
| Lev | Missing dlcq/dlttq → 0 before division |
| TobinsQ | Division by atq; atq=0 yields NaN (inf replaced) |
| ROA | avg_assets in denominator; protected |

### Inf/NaN Handling

- All `inf`/`-inf` values replaced with `NaN` before regression (`run_h8_political_risk.py:226`)
- `dropna(subset=required_avail)` before PanelOLS estimation

## J. Known Issues / Ambiguities

### J.1) Low style_frozen Coverage (62.8%)

**Issue:** Only 18,439 of 29,343 firm-years have valid style_frozen values.

**Cause:** style_frozen requires:
1. CEO matched to calls in manifest (ceo_id column)
2. CEO has ≥5 total calls (Phase 56 filter)
3. Dominant CEO identified for firm-year (≥1 call in frozen window)

**Impact:** 37.2% of firm-years dropped from regression due to missing style_frozen.

**Recommendation:** Document sample selection; consider robustness with lower min_calls threshold.

### J.2) style_frozen Mean Slightly Negative (-0.10)

**Issue:** Expected mean ≈ 0 (standardized), observed mean = -0.10.

**Cause:** style_frozen is standardized ClarityCEO, but assigned at firm-year level via frozen constraint. The subset of firm-years with matched CEOs may differ from full CEO population.

**Impact:** Minor; does not affect interpretation (SD ≈ 1 as expected).

**Status:** ACCEPTABLE — within expected sampling variation.

### J.3) No Forward-Fill for PRiskFY

**Issue:** PRiskFY requires ≥2 quarters in 366-day window; no interpolation.

**Cause:** By design in `_hassan_engine.py` (MIN_QUARTERS=2, no forward-fill).

**Impact:** 6.3% of firm-years missing PRiskFY (1,842 / 29,343).

**Status:** BY DESIGN — conservative approach avoids measurement error from sparse data.

### J.4) Interaction Term Coverage (61.1%)

**Issue:** Only 17,930 firm-years have valid interact term.

**Cause:** Interaction requires both PRiskFY AND style_frozen to be non-missing.

**Impact:** Final regression sample is 15,721 (additional drops from controls/DV).

**Status:** DOCUMENTED — reflects joint data availability requirements.

### J.5) H8 Not Supported (β3 ≈ 0)

**Issue:** Interaction coefficient not significant (p=0.776).

**Interpretation:** CEO speech vagueness does NOT significantly moderate the political risk → abnormal investment relationship in this sample.

**Status:** Meaningful null result — hypothesis test completed as designed.
