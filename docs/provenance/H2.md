## Design Decisions

1. **CashFlow/SalesGrowth excluded:** These are first-stage inputs to Biddle (2009) InvestmentResidual. Including them would create mechanical correlation with the DV.
2. **Extended linguistic controls:** Entire_All_Negative_pct and Analyst_QA_Uncertainty_pct added as domain-specific extended controls.
3. **8 base controls:** InvestmentResidual ∉ {Lev, CashHoldings, CapexAt}, so all three enter as controls.
