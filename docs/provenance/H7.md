# H7: Speech Vagueness and Stock Illiquidity — Provenance Documentation

## Suite Overview

### A1) Identity

| Field | Value |
|-------|-------|
| **Suite ID** | H7 |
| **Research Question** | Does managerial speech vagueness (uncertainty language) predict future stock illiquidity (Amihud 2002)? |
| **Estimation Unit** | Individual earnings call (call-level) |
| **Primary Keys** | `file_name` (unique call identifier), `gvkey` (firm), `year` (time FE) |

### A2) Model Family and Estimator

| Field | Value |
|-------|-------|
| **Model Family** | Panel OLS with Fixed Effects |
| **Estimator** | `linearmodels.PanelOLS` (within transformation) |
| **Objective/Link** | Linear (identity link), OLS estimator |
| **Within Transformation** | Entity-demeaning applied to all variables (absorbs FE) |
| **Weighting** | None (equal-weighted observations) |

### A3) Outcome(s) / Target Definition

| Field | Value |
|-------|-------|
| **Dependent Variable** | `amihud_illiq_lead` |
| **Outcome Coding** | Continuous (Amihud illiquidity ratio × 10^6) |
| **Missingness → Drop** | NaN rows dropped from regression (listwise deletion) |
| **Timing Convention** | Forward-looking: t+1 fiscal year Amihud measure (DV at t predicts IV at t) |

### A4) Main RHS and Controls

| Category | Variables |
|----------|-----------|
| **Key Independent Variables** | `Manager_QA_Uncertainty_pct`, `Manager_Pres_Uncertainty_pct`, `Manager_QA_Weak_Modal_pct` (primary IVs); `CEO_QA_Uncertainty_pct`, `CEO_Pres_Uncertainty_pct`, `CEO_QA_Weak_Modal_pct` (secondary IVs) |
| **Interaction Terms** | None |
| **Control Blocks** | `BASE_CONTROLS`: Size, Lev, ROA, TobinsQ, Volatility, StockRet |
| **Sample Filters** | `min_calls >= 5` per firm within sample; `amihud_illiq_lead` non-missing required |

### A5) Fixed Effects + Variance Estimator

| Field | Value |
|-------|-------|
| **Entity Effects** | Firm (`gvkey`) |
| **Time Effects** | Year (`year` from `start_date`) |
| **Standard Errors** | Clustered by entity (firm) |
| **Cluster Type** | One-way cluster (Moulton correction) |
| **Small-Sample Corrections** | None (default linearmodels behavior) |

### A6) Model-Family Addendum

**N/A** — H7 uses standard panel OLS, not survival/hazards, IV, DiD, or time-series models.

**Implementation** (`winsorization.py:23-84`):

**Variables Winsorized**:
- **CompustatEngine**: Size, Lev, ROA, TobinsQ, CurrentRatio, RD_Intensity, CashHoldings (per-year 1%/99%)
- **CRSPEngine**: StockRet, MarketRet, Volatility, amihud_illiq (per-year 1%/99%)
- **LinguisticEngine**: All 25 `_pct` columns (per-year 0%/99% — upper bound only)

**Note**: The DV (`amihud_illiq_lead`) is NOT re-winsorized because it is a shifted variable computed after panel construction.

## J. Known Issues / Ambiguities (Actionable)

### Issue 1: CEO IVs Have Lower Coverage Than Manager IVs

**Description**: CEO uncertainty variables have ~68% coverage vs ~96% for Manager variables. This is because CEO identification requires exact name matching (speaker_name matches ceo_name or ceo_last), In some calls, the CEO may not speak or may not be correctly identified.

**Impact**: Lower effective sample size for CEO coefficients; potential selection bias if CEO identification is correlated with firm characteristics.

**Recommended Fix**: Document the coverage differential explicitly; consider robustness check using only Manager variables.

**Verification Status**: VERIFIED via coverage check (76,818 / 112,968 = 68.0% for CEO_QA_Uncertainty_pct).

### Issue 2: All β Coefficients Are Negative (Opposite to H7 Direction)

**Description**: All 9 regression models produce negative β1 coefficients for Manager uncertainty variables. H7 predicts β > 0 (vagueness → higher illiquidity). The results consistently show β < 0 or β ≈ 0.

**Impact**: H7 is NOT SUPPORTED by the data. The hypothesis that managerial vagueness increases stock illiquidity is rejected.

**Recommended Fix**: This is a finding, not a bug. Report as "H7 NOT SUPPORTED" with the negative coefficients as evidence.

**Verification Status**: VERIFIED via model_diagnostics.csv review (all h7_sig = False, all beta1 ≤ 0 or p > 0.05).

### Issue 3: Within-R² Is Very Low (0.7%–12%)

**Description**: The within-R² values range from 0.007 to 0.12, indicating that the model explains very little of the within-firm variation in illiquidity.

**Impact**: Even if H7 were supported, the economic magnitude of the effect would be small.

**Recommended Fix**: This is expected for financial panel data. The low R² does not invalidate the hypothesis test, but should be noted in interpretation.

**Verification Status**: VERIFIED via model_diagnostics.csv review.

### Issue 4: Utility Sample Has Very Few Firms (N=78)

**Description**: The Utility sample has only 78 unique firms and ~2,200-2,300 observations after min-calls filtering.

**Impact**: Low statistical power; results should be interpreted with caution.

**Recommended Fix**: Consider merging Utility with Main or excluding from primary analysis.

**Verification Status**: VERIFIED via model_diagnostics.csv review (n_firms = 78 for Utility).
