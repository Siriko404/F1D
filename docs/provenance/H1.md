## A. Suite Overview

### A1) Identity

| Field | Value |
|-------|-------|
| Hypothesis suite name/ID | H1 -- Speech Uncertainty and Cash Holdings |
| Research question | Does managerial speech uncertainty on earnings calls predict higher corporate cash holdings (precautionary savings)? |
| Estimation unit | Individual earnings call (`file_name`) |
| Primary keys | `gvkey` (entity) + `fyearq_int` (time index, Compustat fiscal year) |

### A2) Model Family and Estimator

| Field | Value |
|-------|-------|
| Model family | Panel FE (OLS with fixed effects); two FE configurations tested |
| Estimator | `linearmodels.panel.PanelOLS`, `drop_absorbed=True` |
| FE config 1 (Industry) | Industry FE (FF12, absorbed via `other_effects`) + Year FE (`TimeEffects`); no entity FE |
| FE config 2 (Firm) | Entity FE (`EntityEffects`) + Year FE (`TimeEffects`) via `from_formula` |
| Within transformation | Automatic via PanelOLS (demeaning within absorbed groups) |
| Optimizer/solver | Default `linearmodels` OLS solver |
| Weighting | None |
| Regularization | None |

### A3) Outcome / Target Definition

| Field | Value |
|-------|-------|
| Dependent variables (2) | `CashHoldings` (contemporaneous, t) and `CashHoldings_lead` (next-fiscal-year, t+1) |
| Construction | `cheq / atq` from Compustat. For lead: aggregated to firm-fiscal-year (last call per `(gvkey, fyearq_int)`), then shifted by -1 within `gvkey`. Validated for fiscal-year continuity: non-consecutive fiscal years receive NaN. |
| Timing convention | Contemporaneous (t): Compustat value matched to call date. Forward-looking (t+1): cash holdings at end of next fiscal year. |
| Missing policy | Drop (listwise deletion at Stage 4) |
| Boundary values | Winsorized at 1%/99% by fiscal year at Compustat engine level. |
| Moulton note | Contemporaneous `CashHoldings` is constant within firm-quarter — multiple calls per firm-quarter share the same DV value. Firm-clustered SEs partially address this. Results from cols 1-4 should be interpreted alongside lead DV cols 5-8. |

### A5) Fixed Effects + Variance Estimator

| Field | Value |
|-------|-------|
| Industry FE (cols 1,3,5,7) | FF12 industry classification, absorbed via PanelOLS `other_effects` parameter (10 groups in Main sample) |
| Firm FE (cols 2,4,6,8) | `gvkey` via `EntityEffects` in PanelOLS formula |
| Time FE (all cols) | `fyearq_int` (Compustat fiscal year) via `TimeEffects` |
| Standard errors | Firm-clustered (`cov_type="clustered"`, `cluster_entity=True`) in all 8 specs |
| Justification | DV is constant within (gvkey, fyearq) by construction. Firm-clustered SEs correct for intra-cluster correlation (Moulton problem). |
| Hypothesis test | One-tailed: H1 beta > 0 (higher uncertainty → more cash hoarding) |
| Stars | `***` p<0.01, `**` p<0.05, `*` p<0.10 (one-tailed) |

## G. Outliers, Missing Data, Scaling, and Model-Specific Data Rules

### Winsorization

| Variable group | Method | Thresholds | When applied |
|---------------|--------|------------|--------------|
| Compustat controls (Lev, Size, TobinsQ, ROA, CapexAt, OCF_Volatility, SalesGrowth, RD_Intensity, CashFlow) | By fiscal year (`fyearq`) | 1% / 99% | At engine level, before merge |

| Clarity residuals | Not winsorized (regression residuals) | N/A | By design (mean ≈ 0) |

### Missing Data Policy

- **Listwise deletion** at Stage 4: `prepare_regression_data()` drops all rows where ANY of the 6 IVs or controls is NaN.
- **inf -> NaN** replacement before complete-case filtering.
- **CEO_Clarity_Residual** is the binding constraint: 43.8% coverage in Main sample drives ~58% attrition.
- **Coverage check**: runner warns if any IV has >50% NaN before filtering.

### Standardization

**No standardization is implemented.** All variables enter regressions in natural units.
