## 1. Suite Overview

### 1.1 Identity

| Field | Value |
|-------|-------|
| Hypothesis suite name/ID | H1 -- Speech Uncertainty and Cash Holdings |
| Research question | Does managerial speech uncertainty on earnings calls predict higher corporate cash holdings (precautionary savings)? |
| Estimation unit | Individual earnings call (`file_name`) |
| Primary keys | `gvkey` (entity) + `fyearq_int` (time index, Compustat fiscal year) |
| Prediction | Higher uncertainty language --> higher cash holdings (beta > 0) |

### 1.2 Model Family and Estimator

| Field | Value |
|-------|-------|
| Model family | Panel FE (OLS with fixed effects); two FE configurations tested |
| Estimator | `linearmodels.panel.PanelOLS`, `drop_absorbed=True` |
| FE config 1 (Industry) | Industry FE (FF12, absorbed via `other_effects`) + Year FE (`TimeEffects`); no entity FE |
| FE config 2 (Firm) | Entity FE (`EntityEffects`) + Year FE (`TimeEffects`) via `from_formula` |
| Within transformation | Automatic via PanelOLS (demeaning within absorbed groups) |
| Solver | Default `linearmodels` OLS solver |
| Weighting | None |
| Regularization | None |

### 1.3 Outcome / Target Definition

| Field | Value |
|-------|-------|
| Dependent variables (2) | `CashHoldings` (contemporaneous, t) and `CashHoldings_lead` (next-fiscal-year, t+1) |
| Construction | `cheq / atq` from Compustat (`_compustat_engine.py` line 1066). For lead: aggregated to firm-fiscal-year (latest call per `(gvkey, fyearq_int)`), then shifted by -1 within `gvkey`. Validated for fiscal-year continuity: non-consecutive fiscal years receive NaN (`build_h1_cash_holdings_panel.py` lines 330-334). |
| Timing convention | Contemporaneous (t): Compustat value matched to call date via `merge_asof`. Forward-looking (t+1): cash holdings at end of next fiscal year. |
| Missing policy | Listwise deletion at Stage 4 (`prepare_regression_data()` lines 264-267) |
| Boundary values | Winsorized at 1%/99% by fiscal year at Compustat engine level (`_winsorize_by_year` applied in `_compute_and_winsorize`) |
| Moulton note | Contemporaneous `CashHoldings` is constant within firm-quarter -- multiple calls per firm-quarter share the same DV value. With ~35 calls per firm on average (57,216 obs / 1,615 firms), effective N is substantially lower than reported N. Firm-clustered SEs partially address this. Results from cols 1-4 should be interpreted alongside lead DV cols 5-8. |

### 6.1 Dependent Variables

#### 6.1.1 `CashHoldings` (contemporaneous, t)

| Field | Value |
|-------|-------|
| Paper label | Cash Holdings_t |
| Type | Continuous ratio |
| Raw fields | `cheq` (cash and short-term investments, quarterly), `atq` (total assets, quarterly) |
| Raw data source | `inputs/comp_na_daily_all/comp_na_daily_all.parquet` (Compustat North America quarterly) |

**Complete construction chain:**

1. **Load:** `CompustatEngine.get_data()` reads raw Compustat parquet with `REQUIRED_COMPUSTAT_COLS` including `cheq` and `atq` (`_compustat_engine.py` L1238).
2. **Type coercion:** `gvkey` zero-padded to 6 chars (L1239); `datadate` to datetime (L1240); all numeric fields coerced to float64 (L1243-1244); `fyearq`, `fqtr` coerced to numeric (L1245-1246).
3. **Dedup:** `(gvkey, datadate)` deduplicated keeping last row (most recent restatement) (L1250-1251).
4. **Compute:** `comp["CashHoldings"] = comp["cheq"] / comp["atq"]` (L1066). Division by zero produces `inf`.
5. **Inf cleanup:** `comp["CashHoldings"].replace([np.inf, -np.inf], np.nan)` (L1159, L1171-1172).
6. **Winsorization:** `_winsorize_by_year(comp["CashHoldings"], comp["fyearq"])` at 1%/99% per fiscal year, min 10 obs per year-group (L1194-1201). Uses `quantile(0.01)` and `quantile(0.99)` within each fyearq group, then `clip(lower=p1, upper=p99)` (L443-466).
7. **Cache:** Result cached in `CompustatEngine._cache` singleton (L1264).
8. **Match to manifest:** `CashHoldingsBuilder.build()` calls `engine.match_to_manifest(manifest, root_path)` (`cash_holdings.py` L41). Inside `match_to_manifest()` (`_compustat_engine.py` L1268-1302): manifest sorted by `start_date`, Compustat sorted by `datadate`, then `pd.merge_asof(manifest, comp, left_on="start_date", right_on="datadate", by="gvkey", direction="backward")`. This matches each call to the most recent Compustat quarterly filing on or before the call date.
9. **Extract:** Builder returns `merged[["file_name", "CashHoldings"]]` (`cash_holdings.py` L43).
10. **Panel merge:** `build_h1_cash_holdings_panel.py` merges on `file_name`, `how="left"`, zero row-delta enforced (L209-217).

#### 6.1.2 `CashHoldings_lead` (forward-looking, t+1)

| Field | Value |
|-------|-------|
| Paper label | Cash Holdings_{t+1} |
| Type | Continuous ratio |
| Raw fields | Same as CashHoldings (`cheq / atq`), but from the next fiscal year |

**Complete construction chain:**

Steps 1-10 above produce `CashHoldings` at call level. Then:

11. **Attach fyearq:** `create_lead_variable()` calls `attach_fyearq(panel, root_path)` (`build_h1_cash_holdings_panel.py` L280). Inside `attach_fyearq()` (`panel_utils.py` L76-192): file_name uniqueness asserted (L119-128); `CompustatEngine.get_data()` provides `(gvkey, datadate, fyearq)` sorted by datadate; `pd.merge_asof(panel_sorted, fyearq_df, left_on="_start_date_dt", right_on="datadate", by="gvkey", direction="backward")` (L152-159); result mapped back via `panel["fyearq"] = panel["file_name"].map(fyearq_map)` (L164); match rate must be >= 80% (L170-178).
12. **fyearq_int:** `panel["fyearq_int"] = pd.to_numeric(panel["fyearq"], errors="coerce")` (L293).
13. **Aggregate to firm-fiscal-year:** `groupby(["gvkey", "fyearq_int"])["start_date_dt"].idxmax()` selects the call with the latest start_date within each (gvkey, fiscal year) (L309). CashHoldings from that call is taken as the end-of-fiscal-year proxy (L310-312).
14. **Shift:** Sort by `(gvkey, fyearq_grp)`, then `shift(-1)` within gvkey to get next fiscal year's value (L320-328).
15. **Consecutive-year validation:** `consecutive = fyearq_lead == fyearq_grp + 1`; lead set to NaN if fiscal years are not consecutive (L331-334). This prevents matching across fiscal-year gaps.
16. **Merge back:** Lead values merged back to call level on `(gvkey, fyearq_int)`, `how="left"`, zero row-delta enforced (L360-367).
17. **Result:** Calls in the last fiscal year per firm, or with fiscal-year gaps, receive NaN lead. No additional winsorization applied (inherits from CashHoldings winsorization at step 6).

All four textual uncertainty variables share the same upstream construction pipeline. The differences are the speaker filter and section filter applied at Step 2.2.

#### Shared upstream pipeline (Steps A-F)

**A. Dictionary loading (Stage 2.1, `tokenize_transcripts.py` L768-812):**

The Loughran-McDonald (LM) Master Dictionary (`inputs/Loughran-McDonald_MasterDictionary_1993-2024.csv`) is loaded. Words are uppercased. Seven sentiment categories are extracted: Negative, Positive, **Uncertainty**, Litigious, Strong_Modal, Weak_Modal, Constraining. For each category, a set of words with count > 0 in that category column is built. The vocabulary list is the sorted union of all categorized words.

**B. Tokenization (Stage 2.1, `tokenize_transcripts.py` L901-984):**

Each speaker-turn row from `inputs/Earnings_Calls_Transcripts/speaker_data_{year}.parquet` is tokenized using `sklearn.CountVectorizer` with `token_pattern=r"(?u)\b[a-zA-Z]+\b"` and `lowercase=False` (text is uppercased beforehand at L930). The vectorizer vocabulary is restricted to the LM dictionary words. For each LM category, the count is the sum of the sparse matrix columns corresponding to that category's word set: `result[f"{cat}_count"] = np.array(X[:, indices].sum(axis=1)).flatten()` (L952). Total tokens are counted using the same regex pattern: `raw_text.apply(lambda x: len(regex.findall(x)))` (L959). Output: one row per speaker-turn with `file_name`, `context` (pres/qa), `role`, `employer`, `speaker_name`, `{Category}_count`, `total_tokens`. Saved as `linguistic_counts_{year}.parquet`.

**C. Speaker flagging (Stage 2.2, `build_linguistic_variables.py` L411-464):**

Each speaker-turn is flagged with boolean columns: `is_analyst` (role contains "analyst"), `is_operator` (role contains "operator"), `is_manager` (not analyst, not operator, AND either role matches manager keywords from `inputs/Manager_roles/managerial_roles_extracted.txt` OR employer matches company name), `is_ceo` (speaker name matches `ceo_name` or `prev_ceo_name` from Execucomp-linked manifest, using exact match or last-name match).

**D. Section boundaries:** The `context` column in speaker_data parquet files records whether each speaker-turn belongs to the presentation section (`"pres"`) or the Q&A section (`"qa"`). These are determined by the transcript parser during ingestion.

**E. Aggregation to call-level percentage (Stage 2.2, `build_linguistic_variables.py` L472-502):**

For each (sample, context) combination, the function `aggregate_weighted()` filters to matching speaker-turns, groups by `file_name`, sums `{Category}_count` and `total_tokens`, then computes:

```
{Sample}_{Context}_{Category}_pct = (sum of {Category}_count) / (sum of total_tokens) * 100
```

NaN is preserved for calls where no matching speaker-turns exist (e.g., no CEO identified for CEO variables). Zero total_tokens produces NaN via `.replace(0, np.nan)` (L490).

Output: `linguistic_variables_{year}.parquet` with one row per call (keyed on `file_name`).

**F. Winsorization at LinguisticEngine level (`_linguistic_engine.py` L254-258):**

`LinguisticEngine.get_data()` loads all year-partitioned parquet files, concatenates them, then applies per-year **upper-only** winsorization:

```python
winsorize_by_year(combined, existing_pct_cols, year_col="year",
                  lower=0.0, upper=0.99, min_obs=10)
```

This clips each `_pct` column at the **0th percentile (no lower clipping)** and the **99th percentile** within each calendar year. The `winsorize_by_year()` function (`winsorization.py` L23-84) replaces inf with NaN, then for each year group with >= 10 valid observations, computes `quantile(0.0)` and `quantile(0.99)` and clips. This is a 0%/99% UPPER-ONLY winsorization, not symmetric 1%/99%.

#### 6.3.3 `TobinsQ` (Tobin's Q)

| Field | Value |
|-------|-------|
| Paper label | Tobin's Q |
| Type | Continuous ratio |
| Raw fields | `cshoq` (common shares outstanding, quarterly), `prccq` (price close, quarterly), `dlcq` (short-term debt), `dlttq` (long-term debt), `atq` (total assets) |

**Construction:**
1. `mktcap = comp["cshoq"] * comp["prccq"]` (L1067).
2. `debt_c = comp["dlcq"].clip(lower=0).fillna(0)` (L1068). Negative debt clipped to zero, then NaN treated as zero.
3. `debt_t = comp["dlttq"].clip(lower=0).fillna(0)` (L1069). Same treatment.
4. `debt_book = np.where(comp["dlcq"].isna() & comp["dlttq"].isna(), np.nan, debt_c + debt_t)` (L1070-1072). If BOTH debt fields are NaN, debt_book is NaN (not zero).
5. `comp["TobinsQ"] = np.where(comp["atq"].notna() & (comp["atq"] > 0) & mktcap.notna(), (mktcap + debt_book) / comp["atq"], np.nan)` (L1073-1077). Requires non-null positive atq and non-null mktcap.
6. Inf replaced with NaN (L1171-1172).
7. Winsorized at 1%/99% per fyearq (L1194-1201).

## 11. Identification & Inference Assessment

### 11.1 Identification Strategy

This suite tests a **descriptive/associational** relationship between managerial speech uncertainty and corporate cash holdings. There is no instrumental variable, no exogenous shock, and no regression discontinuity design. The identification relies on conditional correlations within fixed effects structures.

**Causal interpretation is not claimed and not supported.** The design cannot rule out:
- **Reverse causality:** Firms holding more cash may create contexts where managers speak with more uncertainty (e.g., justifying precautionary holdings).
- **Omitted variable bias:** Unobserved firm-level characteristics (e.g., managerial risk aversion, corporate culture) may drive both speech patterns and cash policies simultaneously.

### 11.2 Fixed Effects Interpretation

| FE structure | What it controls for | What survives | Interpretation strength |
|-------------|---------------------|---------------|------------------------|
| Industry FE (FF12) + Year FE | Industry-level and time-level confounds | Between-firm and within-firm variation within industries | Weaker for causal claims (Mundlak 1978) |
| Firm FE + Year FE | All time-invariant firm-level confounds | Within-firm variation over time only | Stronger for causal claims but lower power |

The pattern where Mgr Pres Unc is significant under Industry FE but not Firm FE suggests the effect operates through between-firm variation within industries. This is econometrically weaker for causal identification -- it may reflect persistent firm characteristics correlated with both uncertainty language and cash holdings.

The pattern where Mgr QA Unc is significant under Firm FE but not Industry FE suggests a within-firm effect: when a firm's Q&A uncertainty language increases, its cash holdings increase. This is more suggestive of a real economic relationship, though still subject to time-varying confounders.

### 11.3 Standard Errors

Firm-clustered SEs are appropriate for this panel structure. However:
- The panel is unbalanced (5-67 obs/firm, avg 35). Clustered SEs with many small clusters can be downward-biased (Cameron, Gelbach, and Miller 2008), though with 1,615 clusters this concern is modest.
- Double-clustering by firm and time (Petersen 2009) is not implemented as a robustness check.

### 11.4 Multiple Testing

32 hypothesis tests (4 IVs x 8 specs) are conducted without formal multiple-testing correction. At nominal alpha = 0.05 (one-tailed), the expected number of false positives under the global null is 1.6. The observed 6 significant results (2 Mgr QA + 4 Mgr Pres) exceeds this but is not overwhelming. No Bonferroni, Holm, or FDR correction is applied. This should be discussed in the thesis.

### 11.5 Effective N Concerns

With 57,216 observations from only 1,615 firms over 18 time periods, the average firm contributes ~35 calls. The contemporaneous DV (CashHoldings) is constant within firm-quarter, meaning multiple calls within the same quarter share the identical DV value. This inflates the apparent precision of estimates. Firm-clustered SEs partially address intra-cluster correlation, but the Moulton (1999) problem means the effective sample size is much lower than 57,216. A firm-year-level collapse robustness check would be informative.

## 14. Design Decisions & Rationale

| # | Decision | Rationale |
|---|----------|-----------|
| 1 | **4 simultaneous IVs** (not one-at-a-time) | Horse-racing approach tests which uncertainty dimension has independent explanatory power after controlling for the others. Reflects the 2x2 decomposition (CEO/Manager x QA/Pres). Prior 6-IV specification included clarity residuals (CEO_Clarity_Residual, Manager_Clarity_Residual) but these were removed: they had ~57% missingness driven by Execucomp coverage, causing severe attrition (N ~37k vs ~57k with 4 IVs). |
| 2 | **Industry FE vs Firm FE** comparison | Tests whether effects operate through between-firm or within-firm variation. Firm FE is stronger for causal identification but lower power. Divergent significance patterns across FE structures are informative about the economic channel. |
| 3 | **Contemporaneous + lead DV** | Contemporaneous tests association; lead tests predictive power. Lead DV partially addresses timing concerns (uncertainty at t predicting cash at t+1). |
| 4 | **Main sample only** (excl Finance/Utility) | Standard in corporate finance. Financial firms hold cash for regulatory reasons; utility firms have regulated capital structures. Both confound the precautionary savings hypothesis. |
| 5 | **One-tailed test (beta > 0)** | The precautionary savings hypothesis makes a directional prediction: higher uncertainty --> more cash hoarding. One-tailed test is appropriate for a directional hypothesis. |
| 6 | **Fiscal year (fyearq_int) as time index** | Corrects a prior bug where calendar year was used. ~30% of sample firms have non-December fiscal year-ends. Using `fyearq_int` ensures time FE align with the Compustat data generating process. |
| 7 | **Call-level analysis** (not firm-year) | Preserves maximum information from transcript-level linguistic measures. Different calls within a firm-year may have different uncertainty content. The trade-off is the Moulton problem for contemporaneous DV (see L4). |
| 8 | **Firm-clustered SEs** | Standard for panel data with persistent firm-level unobservables. Accounts for serial correlation within firm over time. |
| 9 | **Min 5 calls per firm** | Ensures each firm contributes enough observations for meaningful within-firm variation under Firm FE. Prevents singleton firms from affecting estimates. |
| 10 | **`drop_absorbed=True`** | PanelOLS parameter that drops variables perfectly collinear with absorbed effects. Prevents rank deficiency errors from singleton groups. |
