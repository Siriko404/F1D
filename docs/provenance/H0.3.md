## A. Suite Overview

### A1) Identity

| Field | Value |
|---|---|
| **Suite name / ID** | H0.3 — CEO Clarity Extended Controls Robustness (Table 4.1.2) |
| **Research question** | Are CEO fixed-effect estimates of Q&A uncertainty stable when augmenting baseline financial controls with extended firm-level controls (Size, BM, Leverage, ROA, Current Ratio, R&D Intensity, Volatility)? |
| **Estimation unit** | Earnings call (one row per call); call-level OLS |
| **Primary keys** | `file_name` (unique per call), `ceo_id` (CEO identifier for FE) |

### A2) Model Family and Estimator

| Field | Value |
|---|---|
| **Model family** | OLS with categorical dummy variables as fixed effects |
| **Estimator** | `statsmodels.formula.api.smf.ols` with `C(ceo_id)` + `C(year)` dummies |
| **Absorption** | Not absorbed — CEO and year FE estimated as explicit dummy coefficients via Patsy `C()` operator |
| **Optimizer** | Default statsmodels OLS (QR decomposition); no explicit solver selection |
| **Weighting** | None |
| **Regularization** | None |

### A3) Outcome / Target Definition

| Field | Value |
|---|---|
| **Dependent variables** | (1) `Manager_QA_Uncertainty_pct`: % uncertainty words in all-manager Q&A speech; (2) `CEO_QA_Uncertainty_pct`: % uncertainty words in CEO-only Q&A speech |
| **Outcome coding** | Continuous percentage (0–100 scale, typically 0–3%). Missingness → row dropped via complete-case filter |
| **Timing convention** | Contemporaneous: DV measured at same call as controls. No lead/lag structure |

### A4) Main RHS and Controls

| Block | Variables |
|---|---|
| **Linguistic controls (Manager models)** | `Manager_Pres_Uncertainty_pct`, `Analyst_QA_Uncertainty_pct`, `Entire_All_Negative_pct` |
| **Linguistic controls (CEO models)** | `CEO_Pres_Uncertainty_pct`, `Analyst_QA_Uncertainty_pct`, `Entire_All_Negative_pct` |
| **Base firm controls** | `StockRet`, `MarketRet`, `EPS_Growth`, `SurpDec` |
| **Extended firm controls** | `Size`, `BM`, `Lev`, `ROA`, `CurrentRatio`, `RD_Intensity`, `Volatility` |
| **Sample filters** | (1) Main industry sample only (all firms except FF12=11 Finance and FF12=8 Utility, including NaN ff12_code; `panel_utils.py:67–73`); (2) CEOs with ≥ 5 calls in the estimation-ready sample; (3) Complete cases across all model variables |

### A5) Fixed Effects and Variance Estimator

| Field | Value |
|---|---|
| **Fixed effects** | CEO FE via `C(ceo_id)` dummies + Year FE via `C(year)` dummies |
| **Standard errors** | Clustered at CEO level (`cov_type="cluster"`, `cov_kwds={"groups": df_reg["ceo_id"]}`) |
| **Small-sample corrections** | None beyond statsmodels default df adjustment for clustered SEs |

### A6) Model-Family Addendum

N/A — OLS; no survival, IV, DiD, RD, time-series, or GLM components.

## G. Outliers, Missing Data, Scaling, and Model-Specific Data Rules

### Winsorization

| Variable Group | Method | Thresholds | Grouping | When Applied | Code Reference |
|---------------|--------|------------|----------|--------------|----------------|
| Linguistic _pct variables | Per-year upper-only | 0%/99% (clips top 1% within each year; no lower bound since these are non-negative proportions) | Calendar year | At LinguisticEngine load (before panel build) | `_linguistic_engine.py:255–258` |
| Compustat variables (Size, BM, Lev, ROA, CurrentRatio, RD_Intensity, EPS_Growth) | Per-year symmetric | 1%/99% within each fiscal year (fyearq) | Fiscal year (fyearq) | At CompustatEngine compute (before panel build) | `_compustat_engine.py:1138–1160` |
| CRSP variables (StockRet, MarketRet, Volatility) | Per-year symmetric | 1%/99% within each calendar year | Calendar year | At CRSPEngine compute (before panel build) | `_crsp_engine.py:419–451` |
| IBES dispersion, earnings_surprise_ratio | Pooled symmetric | 1%/99% global | Pooled (all years) | At IbesEngine compute | `_ibes_engine.py:111–114` |
| SurpDec | Not winsorized | N/A (ordinal -5..+5) | N/A | N/A | N/A |

### Missing Data Policy

- **Listwise deletion per model:** Complete-case filter applied per model specification — rows missing ANY required variable are dropped (`prepare_regression_data()` line 278–279)
- **No imputation:** Missing values are never filled for regression variables (except R&D set to 0 when missing, and Leverage debt components set to 0 when missing — standard accounting convention)
- **Sample varies across models:** Manager models and CEO models have different complete-case sets (CEO_QA coverage = 68% vs Manager_QA = 96%)

### Scaling / Standardization

- **Within-model standardization:** The 11 continuous controls (StockRet, MarketRet, EPS_Growth, SurpDec, Size, BM, Lev, ROA, CurrentRatio, RD_Intensity, Volatility) are z-scored `(x - mean) / std` within the estimation sample of each model (`run_h0_3_ceo_clarity_extended.py:345–362`). This is done for numerical stability (prevents SVD convergence issues with large dummy matrices). SurpDec was added to the standardization list on 2026-03-13 — with improved IBES coverage (70.0% vs prior 18.7%), the unstandardized ordinal values (-5..+5) caused SVD convergence failures in the larger Manager_Baseline design matrix.
- **IMPORTANT — standardization asymmetry:** The 3 linguistic controls (Manager/CEO_Pres_Uncertainty_pct, Analyst_QA_Uncertainty_pct, Entire_All_Negative_pct) are NOT z-scored — they remain in raw percentage units. This means coefficients in the same regression table are in different units: SD units for financial/ordinal controls, raw % for linguistic controls.
- **IMPORTANT:** Standardization is within-sample, meaning financial control coefficient magnitudes are in standard-deviation units, not raw units. Table notes state this correctly.

### Denominator Protections

- `CurrentRatio`: `lctq` replaced with NaN when 0 (prevents division by zero)
- `BM`: implicit protection — cshoq×prccq = 0 produces NaN via pandas
- `Size`: `atq ≤ 0` → NaN (not clamped to minimum)
- `EPS_Growth`: `|epspxq_lag| == 0` → NaN
- Inf values systematically replaced with NaN after all ratio computations (`_compustat_engine.py:1114–1133`)

## K. Corporate Finance Thesis Referee Assessment

### K1) Bottom-Line Referee Verdict

This suite implements a **robustness check** for the CEO clarity fixed-effects estimation — it tests whether adding seven extended financial controls changes the estimated CEO fixed effects. As a robustness check rather than a hypothesis test, the evidentiary bar is different from a causal claim. The implementation is clean: correct OLS with appropriate CEO-level clustering, per-year winsorization of all controls, complete-case analysis, and minimum-calls filter. The key finding that R² barely changes between Baseline and Extended models (0.418 → 0.420 for Manager; 0.368 → 0.368 for CEO) supports the robustness claim. However, several design choices merit scrutiny.

**Overall status: THESIS-STANDARD AS IMPLEMENTED** (for its intended purpose as a robustness check)

**Supported interpretation:** Descriptive association — the suite demonstrates that CEO fixed effects in uncertainty regressions are robust to the inclusion of additional firm-level controls. This supports, but does not establish, the thesis that measured "uncertainty style" reflects a genuine CEO personal characteristic rather than a proxy for omitted firm fundamentals.

**Overselling risk:** Low, provided the write-up does not claim that stability under extended controls proves the fixed effects are causal or unconfounded. Many omitted variables (CEO tenure, CEO age, board composition, governance quality) are not in the extended set.
