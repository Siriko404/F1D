# Debug and Investigation Scripts

This directory contains scripts used for debugging and investigation.

## Contents

- **investigations/** - Scripts investigating specific issues (investigate_*.py)
- **verification/** - Verification and check scripts (verify_*.py, check_*.py)
- **debug_*.py** - Debug scripts for various components

## Purpose

These scripts were used to diagnose issues during development. They may be
useful for future debugging but are not part of the active pipeline.
