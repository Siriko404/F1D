# F1D Dataset Schema

This document profiles the final enriched dataset (`f1d_enriched_YYYY.parquet`) generated by the pipeline.

## 1. Core Identifiers
| Variable | Type | Description |
| :--- | :--- | :--- |
| `file_name` | String | Unique transcript identifier (e.g., `10004947_T`). |
| `start_date` | String | Call start datetime (ISO 8601). |
| `business_quarter` | String | Fiscal quarter (e.g., `2017Q2`). |
| `permno` | Float | CRSP Permanent Number (Stock Identifier). |
| `company_name` | String | Company name from transcript metadata. |
| `company_id` | String | Internal company identifier. |
| `cusip` | String | CUSIP identifier (from transcript). |
| `sedol` | String | SEDOL identifier. |
| `isin` | String | ISIN identifier. |
| `company_ticker` | String | Stock ticker symbol. |
| `process_version` | String | Pipeline version string. |
| `had_duplicate_metadata` | Boolean | Flag if metadata had duplicates. |

## 2. Linguistic Measures
| Variable | Type | Description |
| :--- | :--- | :--- |
| **MaQaUnc** | | **Manager Q&A Uncertainty** |
| `total_tokens_maqaunc` | Float | Total words spoken by managers in Q&A. |
| `MaQaUnc_hits` | Float | Count of uncertainty words. |
| `MaQaUnc_pct` | Float | Percentage (`hits / total`). |
| **MaPresUnc** | | **Manager Presentation Uncertainty** |
| `total_tokens_mapresunc` | Float | Total words spoken by managers in Presentation. |
| `MaPresUnc_hits` | Float | Count of uncertainty words. |
| `MaPresUnc_pct` | Float | Percentage (`hits / total`). |
| **AnaQaUnc** | | **Analyst Q&A Uncertainty** |
| `total_tokens_anaqaunc` | Float | Total words spoken by analysts in Q&A. |
| `AnaQaUnc_hits` | Float | Count of uncertainty words. |
| `AnaQaUnc_pct` | Float | Percentage (`hits / total`). |
| **EntireCallNeg** | | **Entire Call Negativity** |
| `total_tokens_entirecallneg` | Int | Total words in the entire call. |
| `EntireCallNeg_hits` | Int | Count of negative words. |
| `EntireCallNeg_pct` | Float | Percentage (`hits / total`). |

## 3. CCM Linkage & Industry
| Variable | Type | Description |
| :--- | :--- | :--- |
| `gvkey` | Float | Compustat Global Company Key. |
| `ccm_conm` | String | Company name from CCM. |
| `ccm_sic` | Float | SIC Code from CCM. |
| `ccm_cusip` | String | CUSIP from CCM. |
| `ccm_linkprim` | String | Link priority (P/C/J/N). |
| `ccm_linktype` | String | Link type (LC/LU/LS/etc.). |
| `link_method` | String | Method used for linking (Tier 1-4). |
| `link_quality_score` | Float | Score indicating match confidence (100=Best). |
| `fuzzy_match_score` | Float | Similarity score for fuzzy matches. |
| `fuzzy_matched_name` | String | Name matched via fuzzy logic. |
| `fuzzy_needs_review` | Object | Flag for fuzzy matches needing review. |
| `ff12_code` | Int | Fama-French 12 Industry Code. |
| `ff12_name` | String | Fama-French 12 Industry Name. |
| `ff48_code` | Int | Fama-French 48 Industry Code. |
| `ff48_name` | String | Fama-French 48 Industry Name. |
