# Backup Files

This directory contains backup copies of files and directories.

## Contents

- **config/** - Configuration file backups (e.g., project.yaml.backup)
- **scripts/** - Compressed archives of 2_Scripts/ directory
- **root/** - Full directory backups from root

## Purpose

These files are kept for rollback capability. The most recent config backup
should be easily accessible. Older backups can be moved here for long-term storage.
